# KanutsananFX - AI-agenttoepassing voor forexhandel

## Overzicht

KanutsananFX is een geavanceerde AI-agentapplicatie voor forexhandel, gebouwd met **React Native** en **Expo**, en biedt naadloze platformonafhankelijke ondersteuning voor zowel **iOS- als Android**-apparaten. Deze applicatie maakt gebruik van geavanceerde AI-mogelijkheden om gebruikers te helpen bij forexhandel en biedt een uitgebreid pakket aan tools en functies die zijn ontworpen voor zowel beginnende als ervaren handelaren. Met ondersteuning voor **42 talen**, inclusief automatische detectie van apparaattaal en handmatige selectie, zorgt KanutsananFX voor een werkelijk mondiale en toegankelijke handelservaring.

## Belangrijkste kenmerken

*   **AI Chat Agent**: een intelligente gespreksagent die gebruikers door handelsstrategieën en marktanalyses leidt.
*   **4 handelsopdrachten**: voer essentiële handelsoperaties rechtstreeks vanuit de app uit.
*   **MetaAPI-integratie**: Naadloze integratie met MetaAPI voor realtime handelsgegevens en -uitvoering.
*   **Multi-LLM-ondersteuning**: Flexibiliteit om te integreren met verschillende grote taalmodellen voor verbeterde AI-mogelijkheden.
*   **Aanpassing van handelsvaardigheden**: Personaliseer AI-handelsvaardigheden zodat deze overeenkomen met individuele handelsstijlen en voorkeuren.
*   **Meertalige gebruikersinterface**: gebruikersinterface beschikbaar in **42 talen**, met automatische taaldetectie en handmatige aanpassing van de instellingen.
*   **Back-up/herstellen**: maak veilig een back-up van uw applicatiegegevens en instellingen en herstel deze.
*   **Ontwikkelaarsinfo**: toegang tot gedetailleerde informatie voor ontwikkelaars en geavanceerde gebruikers.
*   **Gidsen**: uitgebreide handleidingen en tutorials om gebruikers te helpen aan de slag te gaan en de applicatie onder de knie te krijgen.

## Technische stapel

*   **Framework**: React Native + Expo + TypeScript
*   **Navigatie**: Expo-router
*   **Statusbeheer**: React Context API
*   **Lokale opslag**: Asynchrone opslag
*   **Platformen**: iOS en Android

## Installatie

Volg deze stappen om de ontwikkelomgeving voor KanutsananFX in te stellen:

1.  **Kloon de repository**:

    ```bash
    git-kloon https://github.com/kanutsanan1988/KanutsananFX.git
    cd KanutsananFX
    ```

2.  **Installatieafhankelijkheden**:

    ```bash
    npm installeren
    # of
    pnpm installeren
    ```

## De app uitvoeren

Om de applicatie in de ontwikkelingsmodus uit te voeren:

```bash
npx expo start
```

Hierdoor wordt de Expo-ontwikkelingsserver gestart. U kunt vervolgens de Expo Go-app op uw fysieke apparaat of een emulator/simulator gebruiken om de QR-code te scannen en de app uit te voeren.

## Bouwen voor Android en iOS

### Android

*   **Android Preview-APK**:


    ```bash

    ```

*   **iOS Simulator-build**:

    ```bash

    ```

    ```bash

    ```

## Hoe te gebruiken

Gedetailleerde gebruiksinstructies vindt u in de handleidingen van de applicatie. Kortom, na het starten van de app kunt u communiceren met de AI Chat Agent, handelsopdrachten uitvoeren en uw handelsvaardigheden aanpassen via de intuïtieve gebruikersinterface. Zorg ervoor dat uw MetaAPI-integratie is geconfigureerd voor live handelen.

## Meertalige ondersteuning

KanutsananFX biedt uitgebreide meertalige ondersteuning, waarbij de gebruikersinterface beschikbaar is in **42 talen**. De applicatie detecteert automatisch de taal van uw apparaat en stelt deze in als standaardtaal. U kunt uw voorkeurstaal ook handmatig selecteren in het instellingenmenu. De standaard fallback-taal is Thai.
