# KanutsananFX - Aplicație de agent AI de tranzacționare Forex

## Prezentare generală

KanutsananFX este o aplicație puternică de agent AI concepută pentru tranzacționarea Forex, construită cu **React Native** și **Expo**.Oferă suport perfect atât pentru platformele **iOS**, cât și pentru **Android**, oferind comercianților un instrument inteligent și intuitiv pentru a-și gestiona activitățile de tranzacționare.Aplicația se mândrește acum cu suport multilingv extins, cu **42 de limbi** disponibile, cu detectarea automată a limbii dispozitivului și opțiuni de selecție manuală în cadrul setărilor.

## Caracteristici cheie

* **AI Chat Agent**: Interacționează cu un agent inteligent AI pentru a oferi informații și asistență comercială.
* **4 comenzi de tranzacționare**: executați operațiuni de tranzacționare esențiale direct prin aplicație.
* **Integrare MetaAPI**: Conectați-vă fără probleme cu MetaAPI pentru date și execuție de tranzacționare în timp real.
* **Suport multi-LLM**: Utilizați diverse modele de limbă mari pentru capabilități AI îmbunătățite.
* **Personalizarea abilităților de tranzacționare**: personalizați abilitățile de tranzacționare pentru a se potrivi strategiilor individuale.
* **Interfață de utilizare multilingvă**: interfață de utilizator disponibilă în 42 de limbi cu detectare automată și selecție manuală.
* **Backup/Restaurare**: faceți copii de rezervă și restaurați cu ușurință datele aplicației.
* **Informații despre dezvoltatori**: accesați informații detaliate pentru dezvoltatori.
* **Ghiduri**: ghiduri cuprinzătoare pentru a ajuta utilizatorii să înceapă și să maximizeze potențialul aplicației.

## Stack de tehnologie

* **Cadru**: React Native + Expo + TypeScript
* **Navigație**: Router Expo
* **State Management**: API-ul React Context
* **Stocare locală**: AsyncStorage
* **Platforme**: iOS și Android
* **Asistență lingvistică**: 42 de limbi (de rezervă implicită: thailandeză)

## Instalare

Pentru a configura mediul de dezvoltare, urmați acești pași:

1. **Clonează depozitul:**

    ```bash
    git clone https://github.com/your-repo/KanutsananFX.git # Înlocuiți cu adresa URL a depozitului real
    cd KanutsananFX
    ```

2. **Instalați dependențe:**

    ```bash
    instalare npm
    # sau
    instalare pnpm
    ```

## Rularea aplicației

Pentru a rula aplicația în modul de dezvoltare:

```bash
începe expo npx
```

Aceasta va deschide instrumentele pentru dezvoltatori Expo în browser.Apoi puteți alege să rulați aplicația pe:

* Un emulator sau un dispozitiv Android:

    ```bash
    npm rulează Android
    ```

* Un simulator sau dispozitiv iOS:

    ```bash
    npm rulați ios
    ```

## Clădire pentru Android și iOS

### versiuni Android

* **Android Preview APK:**

    ```bash
    npm rulați build:android:apk
    ```

* **Android Production AAB:**

    ```bash
    npm run build:android:production
    ```

### versiuni iOS

* **Previzualizare iOS Build:**

    ```bash
    npm run build:ios
    ```

* **Compilarea simulatorului iOS:**

    ```bash
    npm rulați build:ios:simulator
    ```

* **Versiune de producție iOS:**

    ```bash
    npm run build:ios:production
    ```

## Cum se utilizează

Instrucțiuni detaliate de utilizare vor fi furnizate în ghidurile aplicației.În general, utilizatorii pot interacționa cu agentul de chat AI, pot executa comenzi de tranzacționare și pot personaliza abilitățile de tranzacționare prin interfața multilingvă intuitivă.

## Suport multilingv

KanutsananFX oferă suport pentru **42 de limbi**, asigurând o experiență de utilizator cu adevărat globală.Aplicația detectează automat limba dispozitivului și o setează ca implicită.De asemenea, utilizatorii pot selecta manual limba preferată din meniul de setări.
