# KanutsananFX - Aplikasi Agen AI Perdagangan Forex

## Gambaran Keseluruhan

KanutsananFX ialah aplikasi agen AI perdagangan Forex yang canggih, direka untuk membantu pedagang dalam pasaran kewangan. Dibangunkan menggunakan **Expo / React Native**, aplikasi ini menawarkan sokongan penuh untuk kedua-dua platform **iOS dan Android**, memastikan pengalaman yang lancar merentasi pelbagai peranti.

## Ciri-ciri Utama

*   **Agen Sembang AI**: Berinteraksi dengan agen AI pintar untuk mendapatkan pandangan dan bantuan perdagangan.
*   **4 Arahan Perdagangan**: Laksanakan arahan perdagangan penting terus dari aplikasi.
*   **Integrasi MetaAPI**: Sambungan yang lancar dengan MetaAPI untuk data pasaran masa nyata dan pelaksanaan perdagangan.
*   **Sokongan Multi-LLM**: Fleksibiliti untuk menggunakan pelbagai Model Bahasa Besar (LLM) untuk keupayaan AI yang dipertingkatkan.
*   **Penyesuaian Kemahiran Perdagangan**: Sesuaikan kemahiran agen AI agar sesuai dengan strategi perdagangan individu.
*   **UI Berbilang Bahasa**: Antara muka pengguna yang menyokong pelbagai bahasa untuk khalayak global.
*   **Sandaran/Pulihkan**: Fungsi sandaran dan pemulihan yang mudah untuk data aplikasi anda.
*   **Maklumat Pembangun**: Akses maklumat penting mengenai pembangunan aplikasi.
*   **Panduan**: Panduan komprehensif untuk membantu pengguna memanfaatkan sepenuhnya aplikasi.

## Timbunan Teknologi

*   **Nama Projek**: KanutsananFX
*   **Rangka Kerja**: React Native + Expo + TypeScript
*   **Navigasi**: Expo Router
*   **Pengurusan Keadaan**: React Context API
*   **Penyimpanan**: AsyncStorage
*   **Platform**: iOS dan Android

## Pemasangan

Untuk memulakan pembangunan, klon repositori dan pasang kebergantungan:

```bash
git clone [URL_REPOSITORI_ANDA]
cd KanutsananFX
npm install
# atau
pnpm install
```

## Menjalankan Aplikasi

Untuk menjalankan aplikasi dalam mod pembangunan:

```bash
npx expo start
```

Untuk menjalankan pada emulator/peranti Android:

```bash
npm run android
```

Untuk menjalankan pada emulator/peranti iOS:

```bash
npm run ios
```

## Pembinaan untuk Android dan iOS

Berikut adalah arahan untuk membina aplikasi untuk platform Android dan iOS:

### Android

*   **APK Pratonton Android**:
    ```bash
npm run build:android:apk
    ```
*   **AAB Pengeluaran Android**:
    ```bash
npm run build:android:production
    ```

### iOS

*   **Binaan Pratonton iOS**:
    ```bash
npm run build:ios
    ```
*   **Binaan Simulator iOS**:
    ```bash
npm run build:ios:simulator
    ```
*   **Binaan Pengeluaran iOS**:
    ```bash
npm run build:ios:production
    ```

## Cara Penggunaan

Setelah aplikasi dipasang, anda boleh melancarkannya pada peranti iOS atau Android anda. Ikuti panduan dalam aplikasi untuk mengkonfigurasi agen AI anda, menyambung ke MetaAPI, dan mula menggunakan arahan perdagangan. Gunakan tetapan untuk menyesuaikan pengalaman anda, termasuk pemilihan bahasa.

## Sokongan Berbilang Bahasa

KanutsananFX menyokong **42 bahasa**, memastikan ia boleh diakses oleh khalayak global. Aplikasi ini secara automatik mengesan bahasa peranti anda dan menetapkan bahasa UI yang sesuai. Anda juga boleh memilih bahasa pilihan anda secara manual melalui menu Tetapan. Bahasa sandaran lalai ialah Thai.
