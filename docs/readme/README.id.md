# KanutsananFX - Aplikasi Agen AI Trading Forex

## Ikhtisar

KanutsananFX adalah Aplikasi Agen AI Perdagangan Forex canggih yang dirancang untuk memberdayakan pedagang dengan otomatisasi cerdas dan wawasan pasar yang komprehensif.Dibangun dengan **React Native** dan **Expo**, ia menawarkan kompatibilitas lintas platform yang lancar, mendukung perangkat **iOS dan Android**.Aplikasi ini memanfaatkan AI mutakhir untuk memberikan pengalaman perdagangan yang intuitif dan kuat.

## Fitur Utama

* **Agen Obrolan AI**: Berinteraksi dengan asisten AI cerdas untuk wawasan perdagangan, saran strategi, dan analisis pasar.
* **4 Perintah Perdagangan**: Jalankan operasi perdagangan penting secara langsung melalui aplikasi.
* **Integrasi MetaAPI**: Terhubung secara lancar dengan MetaAPI untuk data pasar real-time dan eksekusi perdagangan.
* **Dukungan Multi-LLM**: Memanfaatkan berbagai Model Bahasa Besar untuk meningkatkan kemampuan AI dan beragam perspektif analitis.
* **Kustomisasi Keterampilan Perdagangan**: Personalisasikan strategi dan parameter perdagangan AI agar sesuai dengan preferensi individu dan profil risiko.
* **UI Multibahasa**: Nikmati antarmuka pengguna yang tersedia dalam **42 bahasa**, dengan deteksi bahasa perangkat otomatis dan opsi pemilihan manual di Pengaturan.
* **Backup/Restore**: Cadangkan dan pulihkan data dan pengaturan aplikasi Anda dengan aman.
* **Info & Panduan Pengembang**: Akses informasi terperinci untuk pengembang dan panduan pengguna yang komprehensif.

## Tumpukan Teknologi

* **Kerangka Kerja**: Bereaksi Asli + Expo + TypeScript
* **Navigasi**: Router Ekspo
* **Manajemen Negara**: React Context API
* **Penyimpanan Lokal**: Penyimpanan Async
* **Platform**: iOS dan Android

## Instalasi

Untuk menyiapkan lingkungan pengembangan, ikuti langkah-langkah berikut:

1. **Kloning repositori**:

```bash
git clone https://github.com/your-repo/KanutsananFX.git
cd KanutsananFX
```

2. **Instal dependensi**:

```bash
npm install
# atau
pnpm install
```

## Menjalankan Aplikasi

Untuk menjalankan aplikasi dalam mode pengembangan:

```bash
npx expo start
```

Ini akan membuka alat pengembang Expo di browser Anda.Anda kemudian dapat memilih untuk menjalankan aplikasi di:

* Emulator/perangkat Android (`npm run android`)
* Simulator/perangkat iOS (`npm run ios`)

## Membangun untuk Android dan iOS

Berikut petunjuk membangun aplikasi untuk berbagai platform dan tujuan:

### Android

* **Pratinjau APK (untuk pengujian)**:

```bash
npm run build:android:apk
```

* **Produksi AAB (untuk Google Play Store)**:

```bash
npm run build:android:production
```

### iOS

* **Pembangunan Pratinjau (untuk pengujian)**:

```bash
npm run build:ios
```

* **Pembuatan Simulator (untuk Simulator iOS)**:

```bash
npm run build:ios:simulator
```

* **Pembuatan Produksi (untuk Apple App Store)**:

```bash
npm run build:ios:production
```

## Cara Menggunakan

Petunjuk penggunaan terperinci akan diberikan di bagian info dan panduan pengembang aplikasi.Fungsi utama termasuk berinteraksi dengan Agen Obrolan AI, menjalankan perintah perdagangan, dan menyesuaikan keterampilan perdagangan.

## Dukungan Multibahasa

KanutsananFX menawarkan dukungan multibahasa yang luas, dengan antarmuka penggunanya tersedia dalam **42 bahasa**.Aplikasi secara otomatis mendeteksi bahasa perangkat dan menetapkannya sebagai bahasa default.Pengguna juga dapat secara manual memilih bahasa pilihan mereka dari menu Pengaturan.Bahasa cadangan default adalah bahasa Thailand.
