# KanutsananFX - ஃபோரெக்ஸ் வர்த்தக AI முகவர் செயலி
## கண்ணோட்டம்
KanutsananFX என்பது **Expo** மற்றும் **React Native** கொண்டு உருவாக்கப்பட்ட உயர்ந்த ஃபோரெக்ஸ் வர்த்தக AI முகவர் செயலி ஆகும், இது **iOS மற்றும் Android** இரு மேடைகளிலும் பிரச்சினையில்லாமல் செயல்பட உறுதி செய்கிறது. இந்த செயலி வர்த்தகர்களுக்கு அறிவுத்திறன் கொண்ட தானியங்கி பணிகள் மற்றும் விரிவான சந்தை பகுப்பாய்வுகளை வழங்குகிறது.
KanutsananFX இப்போது **42 மொழிகள்**-ஐ ஆதரிக்கிறது என்பதை அறிவிப்பதில் பெருமைப்படுகிறோம், இது உண்மையான உலகளாவிய வர்த்தக அனுபவத்தைக் கொடுக்கிறது. செயலி தானாகவே சாதன மொழியை கண்டுபிடித்து, பயனர்களின் விருப்ப மொழியை அமைப்புகள் மெனுவில் இருந்து கையொப்பமிட கூடிய தொழிற்பயிற்சியை வழங்குகிறது.
## முக்கிய அம்சங்கள்
*   **AI பேச்சு முகவர்**: வர்த்தக ஆலோசனை, சந்தை பகுப்பாய்வு மற்றும் நடைமுறை உருவாக்கத்திற்கான அறிவுத்திறனுடைய AI உதவியாளருடன் தொடர்பு கொள்ளுங்கள்.
*   **4 வர்த்தக கட்டளைகள்**: அவசியமான வர்த்தக செயல்களை நேரடியாக செயலியில் நடாத்துங்கள்.
*   **MetaAPI ஒருங்கிணைப்பு**: நேரடி சந்தை தரவு மற்றும் வர்த்தக செயல்பாட்டிற்காக MetaAPI-யுடன் தன்னிறைவான இணைப்பு.
*   **பல LLM ஆதரவு**: மேம்பட்ட AI திறன்களுக்கும் பல்வகை பகுப்பாய்வுக்குமான பல பெரிய மொழி மாதிரிகளை பயன்படுத்தவும்.
*   **வர்த்தக திறன் தனிப்பயன்**: தனிப்பட்ட விருப்பங்கள் மற்றும் ஆபத்து அனுபவங்களை பொருந்தும் வர்த்தக தந்திரங்களை தனிப்பயனாக்கவும் பராமரிக்கவும்.
*   **பல மொழி UI**: 42 மொழிகளில் முழுமையாக உள்ளூர்மொழிபெயர்க்கப்பட்ட பயனர் இடைமுகத்துடன் உலகளாவிய தெளிவான அனுபவம்.
*   **பதிவு/மீட்டமைக்க**: உங்கள் செயலி தரவையும் அமைப்புகளையும் பாதுகாப்பாக பின்பதிவு செய்து மீட்டமைக்க.
*   **முதன்மை தகவல்கள் மற்றும் வழிகாட்டிகள்**: செயலி உபயோகத்தை அதிகரிக்க விரிவான ஆக்கப்பொருட்கள் மற்றும் பயனர் வழிகாட்டிகள்.
## தொழில்நுட்ப தொகுப்பு
*   **திட்டத்தின் பெயர்**: KanutsananFX
*   **வடிவமைப்பு**: React Native + Expo + TypeScript
*   **நெவிகேஷன்**: Expo Router

*   **உள்ளூர் சேமிப்பு**: AsyncStorage
*   **மேடைகள்**: iOS மற்றும் Android
*   **மொழி ஆதரவு**: 42 மொழிகள்
*   **இயல்புநிலை மொழி**: தாய் (Thai)
## நிறுவல்
KanutsananFX டெவலப்ப்மெண்ட் சூழலை அமைக்க, இங்கே உள்ள படிகளை பின்பற்றவும்:
1.  **கிட் ரெப்பாசிடரி நகலை இடுக (_Clone_ செய்ய):**
    ```bash
    git clone <repository_url>
    cd KanutsananFX
    ```
2.  **நிர்வாக இணைப்புகளை நிறுவுக:**
    ```bash
    npm install
    # அல்லது நீங்கள் pnpm விரும்பினால்
    pnpm install
    ```
## செயலியை இயக்கு
செயலி உருவாக்கும் நிலைமையில் இயக்க:
```bash
npx expo start
```
Expo உருவாக்கும் சர்வர் இயங்கும்போது, நீங்கள்:
*   `a` அழுத்தி Android எம்யுலேட்டர்/சாதனத்தில் இயக்கலாம்.
*   `i` அழுத்தி iOS சிமுலேட்டர்/சாதனத்தில் இயக்கலாம்.
மற்றும் பின்வரும் கட்டளைகளையும் பயன்படுத்தலாம்:
```bash
npm run android
npm run ios
```
## Android மற்றும் iOS க்கான கட்டமைப்பு
பல மேடைகளுக்கான மற்றும் விநியோகிக்கும் முறைகளுக்கான செயலி கட்டமைக்கும் கட்டளைகள்:
### Android
*   **முன்னோட்ட APK**: சோதனை மற்றும் முன்னோட்டத்திற்க்காக APK உருவாக்கு.
    ```bash
    npm run build:android:apk
    ```
*   **உற்பத்தி AAB**: Google Play Store விற்கான Android App Bundle (AAB) உற்பத்தி.
    ```bash
    npm run build:android:production
    ```
### iOS
*   **முன்னோட்ட கட்டமைப்பு**: சோதனைக்கான iOS கட்டமைப்பை உருவாக்கவும்.
    ```bash
    npm run build:ios
    ```
*   **சிமுலேட்டர் கட்டமைப்பு**: iOS சிமுலேட்டர்க்கான அலக்கல் கட்டமைப்பு.
    ```bash
    npm run build:ios:simulator
    ```
*   **உற்பத்தி கட்டமைப்பு**: Apple App Store விற்கான iOS உற்பத்தி கட்டமைப்பு.
    ```bash
    npm run build:ios:production
    ```
## பயன்படுத்தும் முறை
KanutsananFX உங்கள் ஃபோரெக்ஸ் வர்த்தக நடவடிக்கைகளை நிர்வகிக்க ஒரு தெளிவான இடைமுகத்தை வழங்குகிறது. செயலியை திறக்கும் பொழுது, நீங்கள்:
1.  **AI பேச்சு முகவருடன் குறியீடு**: கேள்விகள் கேளுங்கள், சந்தை பார்வைகள் பெறுங்கள் மற்றும் வர்த்தக பரிந்துரைகளை பெற்றுக்கொள்ளுங்கள்.
2.  **வர்த்தக கட்டளைகளை செயல்படுத்தவும்**: உள்ளடக்கிய கட்டளைகளை பயன்படுத்தி வர்த்தகங்களை திறக்க, நிறுத்த அல்லது மாற்றவும்.
3.  **வர்த்தக திறன்களை தனிப்பயனாக்கவும்**: AI பராமரிப்புகளை ஒழுங்குபடுத்தி தனிப்பட்ட வர்த்தக தந்திரங்களை அமைக்கவும்.
4.  **மொழி அமைப்புகளை நிர்வகிக்கவும்**: தானாக கண்டறியப்படாதால், செட்டிங்குகளில் இருந்து செயலி மொழியை மாற்றவும்.
5.  **பதிவு மற்றும் மீட்டமைப்பு செய்யவும்**: உங்கள் வர்த்தக அமைப்புகளும் தரவுகளும் பாதுகாப்பாக இருக்க பதிப்பு மற்றும் மீட்டமைப்பு அம்சங்களைப் பயன்படுத்தவும்.
