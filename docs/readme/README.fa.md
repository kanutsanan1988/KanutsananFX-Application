# KanutsananFX - اپلیکیشن ایجنت هوش مصنوعی معاملات فارکس

## معرفی

KanutsananFX یک اپلیکیشن پیشرفته ایجنت هوش مصنوعی است که برای معاملات فارکس طراحی شده است. این برنامه با استفاده از **Expo / React Native** ساخته شده و از هر دو پلتفرم **iOS و Android** پشتیبانی می‌کند. این اپلیکیشن از هوش مصنوعی برای کمک به معامله‌گران در جنبه‌های مختلف معاملات فارکس، با ارائه ابزارهای قدرتمند و تجربه‌ای کاملاً قابل تنظیم، استفاده می‌کند.

## ویژگی‌های کلیدی

*   **ایجنت چت هوش مصنوعی**: برای دریافت بینش و پشتیبانی در معاملات، با یک دستیار هوش مصنوعی هوشمند تعامل کنید.
*   **۴ دستور معاملاتی**: دستورات اصلی معاملاتی را مستقیماً از طریق اپلیکیشن اجرا کنید.
*   **یکپارچه‌سازی با MetaAPI**: برای دریافت داده‌های بازار و اجرای معاملات به صورت لحظه‌ای، به طور یکپارچه به MetaAPI متصل شوید.
*   **پشتیبانی از چند LLM**: از انعطاف‌پذیری یکپارچه‌سازی با مدل‌های زبان بزرگ مختلف بهره‌مند شوید.
*   **سفارشی‌سازی مهارت‌های معاملاتی**: مهارت‌های معاملاتی هوش مصنوعی را مطابق با استراتژی و ترجیحات خود تنظیم کنید.
*   **رابط کاربری چندزبانه**: از رابط کاربری موجود در **۴۲ زبان**، با قابلیت تشخیص خودکار زبان دستگاه و انتخاب دستی زبان در تنظیمات، لذت ببرید.
*   **پشتیبان‌گیری و بازیابی**: به طور امن از داده‌های اپلیکیشن خود پشتیبان تهیه کرده و آن‌ها را بازیابی کنید.
*   **اطلاعات و راهنمای توسعه‌دهندگان**: به اطلاعات و راهنماهای جامع برای توسعه‌دهندگان دسترسی داشته باشید.

## پشته فناوری

*   **نام پروژه**: KanutsananFX
*   **فریم‌ورک**: React Native + Expo + TypeScript
*   **ناوبری**: Expo Router
*   **مدیریت وضعیت**: React Context API
*   **ذخیره‌سازی محلی**: AsyncStorage
*   **پلتفرم‌ها**: iOS و Android
*   **پشتیبانی از زبان**: ۴۲ زبان
*   **زبان پیش‌فرض بازگشتی**: تایلندی

## نصب و راه‌اندازی

برای راه‌اندازی محیط توسعه، مراحل زیر را دنبال کنید:

۱. **کلون کردن مخزن**:

```bash
git clone https://github.com/kanutsanan1988/KanutsananFX.git
cd KanutsananFX
```

۲. **نصب وابستگی‌ها**:

```bash
npm install
# یا
pnpm install
```

## اجرای اپلیکیشن

برای اجرای اپلیکیشن در حالت توسعه:

*   **اجرای سرور توسعه Expo**:

```bash
npx expo start
```

*   **اجرا در شبیه‌ساز/دستگاه اندروید**:

```bash
npm run android
```

*   **اجرا در شبیه‌ساز/دستگاه iOS**:

```bash
npm run ios
```

## ساختن نسخه نهایی برای اندروید و iOS

### نسخه‌های اندروید

*   **APK پیش‌نمایش اندروید**:

```bash
npm run build:android:apk
```

*   **AAB تولیدی اندروید**:

```bash
npm run build:android:production
```

### نسخه‌های iOS

*   **نسخه پیش‌نمایش iOS**:

```bash
npm run build:ios
```

*   **نسخه شبیه‌ساز iOS**:

```bash
npm run build:ios:simulator
```

*   **نسخه تولیدی iOS**:

```bash
npm run build:ios:production
```
