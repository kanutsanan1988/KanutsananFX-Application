# KanutsananFX - Приложение AI-агента для торговли на Форекс

KanutsananFX — это передовое мобильное приложение, разработанное для автоматизации и оптимизации вашей торговли на рынке Форекс с помощью искусственного интеллекта. Приложение поддерживает как **iOS, так и Android** благодаря использованию **Expo / React Native**, обеспечивая кроссплатформенную совместимость и удобство использования.

## Ключевые особенности

*   **AI Chat Agent:** Интеллектуальный чат-агент для взаимодействия и управления торговыми операциями.
*   **4 торговые команды:** Поддержка четырех основных торговых команд для гибкого управления стратегиями.
*   **Интеграция с MetaAPI:** Бесшовная интеграция с MetaAPI для надежного и быстрого исполнения сделок.
*   **Поддержка нескольких LLM:** Возможность использования различных больших языковых моделей (LLM) для повышения эффективности AI.
*   **Настройка торговых навыков:** Гибкая настройка и адаптация торговых навыков AI-агента под ваши нужды.
*   **Многоязычный пользовательский интерфейс:** Интуитивно понятный интерфейс, доступный на нескольких языках.
*   **Резервное копирование и восстановление:** Функции для безопасного резервного копирования и восстановления ваших данных и настроек.
*   **Информация для разработчиков:** Подробная информация и руководства для разработчиков.
*   **Руководства:** Комплексные руководства по использованию приложения и его функций.

## Технологический стек

*   **Фреймворк:** React Native + Expo + TypeScript
*   **Навигация:** Expo Router
*   **Управление состоянием:** React Context API
*   **Хранилище:** AsyncStorage
*   **Платформы:** iOS и Android

## Установка

Для установки зависимостей проекта выполните следующие команды:

```bash
npm install
# или
pnpm install
```

## Запуск приложения

Чтобы запустить приложение в режиме разработки, используйте команду:

```bash
npx expo start
```

После запуска вы можете открыть приложение на своем устройстве с помощью приложения Expo Go или в симуляторе/эмуляторе.

## Сборка для Android и iOS

### Сборка для Android

*   **Предварительный APK:**
    ```bash
npm run build:android:apk
    ```
*   **Промышленный AAB (Android App Bundle):**
    ```bash
npm run build:android:production
    ```

### Сборка для iOS

*   **Предварительная сборка iOS:**
    ```bash
npm run build:ios
    ```
*   **Сборка для симулятора iOS:**
    ```bash
npm run build:ios:simulator
    ```
*   **Промышленная сборка iOS:**
    ```bash
npm run build:ios:production
    ```

## Как использовать

Приложение KanutsananFX предоставляет интуитивно понятный интерфейс для взаимодействия с AI-агентом. После запуска вы можете использовать чат для отправки торговых команд и получения аналитики. Подробные руководства доступны в приложении.

## Многоязычная поддержка

Приложение KanutsananFX поддерживает **42 языка**, обеспечивая глобальный доступ и удобство использования. Оно автоматически определяет язык устройства и позволяет вручную выбрать предпочтительный язык в настройках. Язык по умолчанию, если язык устройства не поддерживается, — тайский.
