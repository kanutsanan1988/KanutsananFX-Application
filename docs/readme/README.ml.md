# KanutsananFX - ഫോറെക്സ് ട്രേഡിംഗ് AI ഏജന്റ് ആപ്ലിക്കേഷൻ

## അവലോകനം
KanutsananFX എന്നത് **React Native**, **Expo**, **TypeScript** എന്നിവ ഉപയോഗിച്ച് നിർമ്മിച്ച ഒരു നൂതന ഫോറെക്സ് ട്രേഡിംഗ് AI ഏജന്റ് ആപ്ലിക്കേഷനാണ്. AI കഴിവുകളും MetaAPI-യുമായുള്ള തടസ്സമില്ലാത്ത സംയോജനവും പ്രയോജനപ്പെടുത്തി, ഓട്ടോമേറ്റഡ് ഫോറെക്സ് ട്രേഡിംഗിനായി ഈ ആപ്ലിക്കേഷൻ ശക്തമായ ഒരു പ്ലാറ്റ്ഫോം നൽകുന്നു. ഇത് ക്രോസ്-പ്ലാറ്റ്ഫോമായി രൂപകൽപ്പന ചെയ്തിട്ടുള്ളതാണ്, **iOS**, **Android** ഉപകരണങ്ങളിൽ സ്ഥിരവും ശക്തവുമായ ഉപയോക്തൃ അനുഭവം നൽകുന്നു.

## പ്രധാന സവിശേഷതകൾ
*   **AI ചാറ്റ് ഏജന്റ്**: ട്രേഡിംഗ് ഉൾക്കാഴ്ചകൾക്കും സഹായത്തിനുമായി ഒരു ബുദ്ധിമാനായ AI ഏജന്റുമായി സംവദിക്കുക.
*   **4 ട്രേഡിംഗ് കമാൻഡുകൾ**: ആപ്ലിക്കേഷൻ വഴി നേരിട്ട് പ്രധാനപ്പെട്ട ട്രേഡിംഗ് പ്രവർത്തനങ്ങൾ നടത്തുക.
*   **MetaAPI സംയോജനം**: തത്സമയ ട്രേഡിംഗ് ഡാറ്റയ്ക്കും നിർവ്വഹണത്തിനുമായി MetaAPI-യുമായി തടസ്സമില്ലാതെ ബന്ധിപ്പിക്കുക.
*   **മൾട്ടി-LLM പിന്തുണ**: മെച്ചപ്പെട്ട AI കഴിവുകൾക്കായി വിവിധ വലിയ ഭാഷാ മോഡലുകൾ ഉപയോഗിക്കുക.
*   **ട്രേഡിംഗ് സ്കിൽ കസ്റ്റമൈസേഷൻ**: വ്യക്തിഗത മുൻഗണനകൾക്ക് അനുസരിച്ച് ട്രേഡിംഗ് തന്ത്രങ്ങൾ വ്യക്തിഗതമാക്കുകയും മികച്ചതാക്കുകയും ചെയ്യുക.
*   **ബഹുഭാഷാ UI**: **42 ഭാഷകളിൽ** ലഭ്യമായ ഒരു ഉപയോക്തൃ ഇന്റർഫേസ് ആസ്വദിക്കുക, ഉപകരണ ഭാഷാ സ്വയമേവ കണ്ടെത്തലും ക്രമീകരണങ്ങളിൽ മാനുവൽ തിരഞ്ഞെടുക്കൽ ഓപ്ഷനുകളും ലഭ്യമാണ്.
*   **ബാക്കപ്പ്/പുനഃസ്ഥാപിക്കുക**: നിങ്ങളുടെ ആപ്ലിക്കേഷൻ ഡാറ്റ സുരക്ഷിതമായി ബാക്കപ്പ് ചെയ്യുകയും പുനഃസ്ഥാപിക്കുകയും ചെയ്യുക.
*   **ഡെവലപ്പർ വിവരങ്ങൾ**: ഡെവലപ്പർമാർക്കായി വിശദമായ വിവരങ്ങൾ ആക്സസ് ചെയ്യുക.
*   **വഴികാട്ടികൾ**: ഉപയോക്താക്കൾക്ക് ആരംഭിക്കാനും ആപ്ലിക്കേഷന്റെ സാധ്യതകൾ പരമാവധി പ്രയോജനപ്പെടുത്താനും സഹായിക്കുന്ന സമഗ്രമായ വഴികാട്ടികൾ.

## ടെക് സ്റ്റാക്ക്
*   **ഫ്രെയിംവർക്ക്**: React Native + Expo + TypeScript
*   **നാവിഗേഷൻ**: Expo Router
*   **സ്റ്റേറ്റ് മാനേജ്മെന്റ്**: React Context API
*   **ലോക്കൽ സ്റ്റോറേജ്**: AsyncStorage
*   **പ്ലാറ്റ്‌ഫോമുകൾ**: iOS, Android

## ഇൻസ്റ്റാളേഷൻ
KanutsananFX പ്രോജക്റ്റ് പ്രാദേശികമായി സജ്ജീകരിക്കുന്നതിന് ഈ ഘട്ടങ്ങൾ പാലിക്കുക:

1.  **റെപ്പോസിറ്ററി ക്ലോൺ ചെയ്യുക**:
    ```bash
    git clone https://github.com/kanutsanan1988/KanutsananFX.git
    cd KanutsananFX
    ```

2.  **ഡിപൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക**:
    ```bash
    npm install
    # അല്ലെങ്കിൽ നിങ്ങൾ pnpm ഇഷ്ടപ്പെടുന്നുവെങ്കിൽ
    pnpm install
    ```

## ആപ്പ് പ്രവർത്തിപ്പിക്കുന്നു
ഡെവലപ്‌മെന്റ് മോഡിൽ ആപ്ലിക്കേഷൻ പ്രവർത്തിപ്പിക്കുന്നതിന്:

```bash
npx expo start
```

ഇത് നിങ്ങളുടെ ബ്രൗസറിൽ Expo ഡെവലപ്പർ ടൂളുകൾ തുറക്കും. തുടർന്ന്, ഒരു iOS സിമുലേറ്ററിലോ, Android എമുലേറ്ററിലോ, അല്ലെങ്കിൽ QR കോഡ് സ്കാൻ ചെയ്തുകൊണ്ട് നിങ്ങളുടെ ഫിസിക്കൽ ഉപകരണത്തിലോ ആപ്പ് പ്രവർത്തിപ്പിക്കാൻ നിങ്ങൾക്ക് തിരഞ്ഞെടുക്കാം.

## Android, iOS എന്നിവയ്‌ക്കായുള്ള നിർമ്മാണം
KanutsananFX Android, iOS പ്ലാറ്റ്‌ഫോമുകൾക്കായി നിർമ്മിക്കുന്നതിനെ പിന്തുണയ്ക്കുന്നു. വിവിധ ബിൽഡ് തരങ്ങൾക്കുള്ള നിർദ്ദേശങ്ങൾ താഴെ നൽകുന്നു.

### Android ബിൽഡുകൾ

*   **Android പ്രിവ്യൂ APK**:
    ```bash
    npm run build:android:apk
    ```

*   **Android പ്രൊഡക്ഷൻ AAB**:
    ```bash
    npm run build:android:production
    ```

### iOS ബിൽഡുകൾ

*   **iOS പ്രിവ്യൂ ബിൽഡ്**:
    ```bash
    npm run build:ios
    ```

*   **iOS സിമുലേറ്റർ ബിൽഡ്**:
    ```bash
    npm run build:ios:simulator
    ```

*   **iOS പ്രൊഡക്ഷൻ ബിൽഡ്**:
    ```bash
    npm run build:ios:production
    ```

## എങ്ങനെ ഉപയോഗിക്കാം
ആപ്ലിക്കേഷന്റെ ഗൈഡുകളിലും ഡോക്യുമെന്റേഷനുകളിലും വിശദമായ ഉപയോഗ നിർദ്ദേശങ്ങൾ നൽകിയിട്ടുണ്ട്. ഉപയോക്താക്കൾക്ക് AI ചാറ്റ് ഏജന്റുമായി സംവദിക്കാനും, ട്രേഡിംഗ് കമാൻഡുകൾ നടപ്പിലാക്കാനും, അവബോധജന്യമായ ബഹുഭാഷാ ഉപയോക്തൃ ഇന്റർഫേസിലൂടെ അവരുടെ ട്രേഡിംഗ് കഴിവുകൾ ഇഷ്ടാനുസൃതമാക്കാനും കഴിയും.
