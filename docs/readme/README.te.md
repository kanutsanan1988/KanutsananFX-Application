# KanutsananFX - Forex ట్రేడింగ్ AI ఏజెంట్ అప్లికేషన్

## అవలోకనం

KanutsananFX అనేది ఒక అధునాతన Forex ట్రేడింగ్ AI ఏజెంట్ అప్లికేషన్, ఇది iOS మరియు Android ప్లాట్‌ఫారమ్‌లకు Expo / React Native ద్వారా మద్దతు ఇస్తుంది. ఈ అప్లికేషన్ 42 భాషలకు మద్దతు ఇస్తుంది, పరికర భాషను స్వయంచాలకంగా గుర్తించగలదు మరియు సెట్టింగ్‌లలో మాన్యువల్‌గా భాషను ఎంచుకునే సౌలభ్యాన్ని అందిస్తుంది.

## ముఖ్య లక్షణాలు

*   **AI చాట్ ఏజెంట్:** మీ ట్రేడింగ్ ప్రశ్నలకు సమాధానం ఇవ్వడానికి మరియు మార్కెట్ విశ్లేషణను అందించడానికి ఒక తెలివైన AI చాట్ ఏజెంట్.
*   **4 ట్రేడింగ్ ఆదేశాలు:** ట్రేడింగ్‌ను సులభతరం చేయడానికి 4 ప్రాథమిక ట్రేడింగ్ ఆదేశాలు.
*   **MetaAPI ఇంటిగ్రేషన్:** MetaAPI ద్వారా నిజ-సమయ మార్కెట్ డేటా మరియు ట్రేడింగ్ అమలు.
*   **బహుళ-LLM మద్దతు:** వివిధ పెద్ద భాషా నమూనాలకు (LLM) మద్దతు.
*   **ట్రేడింగ్ నైపుణ్య అనుకూలీకరణ:** మీ ట్రేడింగ్ శైలికి అనుగుణంగా AI ఏజెంట్ నైపుణ్యాలను అనుకూలీకరించండి.
*   **బహుభాషా UI:** 42 భాషలలో అందుబాటులో ఉన్న వినియోగదారు ఇంటర్‌ఫేస్.
*   **బ్యాకప్/పునరుద్ధరణ:** మీ డేటాను సురక్షితంగా బ్యాకప్ చేయండి మరియు అవసరమైనప్పుడు పునరుద్ధరించండి.
*   **డెవలపర్ సమాచారం:** డెవలపర్‌ల కోసం వివరణాత్మక సమాచారం.
*   **గైడ్‌లు:** అప్లికేషన్‌ను సమర్థవంతంగా ఉపయోగించుకోవడానికి సమగ్ర గైడ్‌లు.

## టెక్ స్టాక్

*   **ప్రాజెక్ట్ పేరు:** KanutsananFX
*   **ఫ్రేమ్‌వర్క్:** React Native + Expo + TypeScript
*   **నావిగేషన్:** Expo Router
*   **స్టేట్ మేనేజ్‌మెంట్:** React Context API
*   **స్టోరేజ్:** AsyncStorage
*   **ప్లాట్‌ఫారమ్‌లు:** iOS మరియు Android
*   **భాషా మద్దతు:** 42 భాషలు
*   **డిఫాల్ట్ ఫాల్‌బ్యాక్ భాష:** థాయ్

## ఇన్‌స్టాలేషన్

ప్రాజెక్ట్‌ను ఇన్‌స్టాల్ చేయడానికి, కింది ఆదేశాలను అమలు చేయండి:

```bash
npm install
```

లేదా pnpm ఉపయోగిస్తుంటే:

```bash

```

## యాప్‌ను అమలు చేయడం

అప్లికేషన్‌ను అభివృద్ధి మోడ్‌లో అమలు చేయడానికి:

```bash
npx expo start
```

Android ఎమ్యులేటర్‌లో అమలు చేయడానికి:

```bash

```

iOS సిమ్యులేటర్‌లో అమలు చేయడానికి:

```bash

```
