# KanutsananFX - Applicazione AI Agent per il Trading Forex

## Panoramica
KanutsananFX è un’applicazione mobile sofisticata progettata per potenziare i trader forex con un agente AI avanzato. Realizzata con **React Native**, **Expo** e **TypeScript**, questa applicazione offre una compatibilità cross-platform fluida, supportando dispositivi **iOS** e **Android**. Integra capacità AI avanzate per assistere gli utenti nell’esecuzione di strategie di trading e nella gestione efficiente delle operazioni forex. L’app vanta un ampio supporto multilingue, offrendo l’interfaccia in **42 lingue** con rilevamento automatico della lingua del dispositivo e opzioni di selezione manuale.

## Caratteristiche Principali
*   **Agente Chat AI**: Interagisci con un agente AI intelligente per approfondimenti e assistenza nel trading.
*   **Comandi di Trading**: Esegui 4 comandi di trading principali direttamente dall’applicazione.
*   **Integrazione MetaAPI**: Integrazione fluida con MetaAPI per dati di trading in tempo reale ed esecuzione.
*   **Supporto Multi-LLM**: Sfrutta diversi Large Language Models per capacità AI diversificate.
*   **Personalizzazione delle Competenze di Trading**: Personalizza le competenze di trading per adattarle alle strategie individuali.
*   **Interfaccia Multilingue**: UI disponibile in 42 lingue, con rilevamento automatico della lingua del dispositivo.
*   **Backup/Ripristino**: Esegui backup e ripristino sicuri dei dati e delle impostazioni dell’app.
*   **Info e Guide per Sviluppatori**: Risorse complete per sviluppatori e utenti.

## Tecnologie Utilizzate
*   **Framework**: React Native + Expo + TypeScript
*   **Navigazione**: Expo Router
*   **Gestione Stato**: React Context API
*   **Archiviazione Locale**: AsyncStorage
*   **Piattaforme**: iOS e Android
*   **Supporto Lingue**: 42 lingue (fallback predefinito: Thai)

## Installazione
Per iniziare con KanutsananFX, segui questi passaggi:

1.  **Clona il repository**:
    ```bash
    git clone <repository_url>
    cd KanutsananFX
    ```

2.  **Installa le dipendenze**:
    ```bash
    npm install
    # oppure
    pnpm install
    ```

## Avvio dell’App
Per avviare l’applicazione in modalità sviluppo:

```bash
npx expo start
```

Questo aprirà gli Expo Developer Tools nel browser, da cui potrai aprire l’app sul tuo dispositivo o simulatore.

## Build per Android e iOS
KanutsananFX supporta diverse configurazioni di build per entrambe le piattaforme Android e iOS.

### Android

*   **APK Anteprima Android**:
    ```bash
    npm run build:android:apk
    ```

*   **AAB Produzione Android**:
    ```bash
    npm run build:android:production
    ```

### iOS

*   **Build Anteprima iOS**:
    ```bash
    npm run build:ios
    ```

*   **Build Simulatore iOS**:
    ```bash
    npm run build:ios:simulator
    ```

*   **Build Produzione iOS**:
    ```bash
    npm run build:ios:production
    ```
