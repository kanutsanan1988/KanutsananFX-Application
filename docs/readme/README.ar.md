# KanutsananFX - تطبيق وكيل الذكاء الاصطناعي لتداول الفوركس

## نظرة عامة
KanutsananFX هو تطبيق متقدم لوكيل الذكاء الاصطناعي لتداول الفوركس، تم بناؤه باستخدام **Expo / React Native**، ويوفر دعمًا سلسًا لكل من منصات **iOS و Android**. يمكّن هذا التطبيق المتداولين من الحصول على مساعدة ذكية من الذكاء الاصطناعي ووظائف تداول قوية.

## الميزات الرئيسية
-   **وكيل الدردشة بالذكاء الاصطناعي**: تفاعل مع وكيل دردشة ذكي بالذكاء الاصطناعي للحصول على رؤى ومساعدة في التداول.
-   **4 أوامر تداول**: نفذ عمليات التداول الأساسية مباشرة من خلال التطبيق.
-   **تكامل MetaAPI**: اتصل بسلاسة مع MetaAPI للحصول على بيانات التداول والتنفيذ في الوقت الفعلي.
-   **دعم LLM متعدد**: استفد من نماذج اللغات الكبيرة المختلفة لقدرات الذكاء الاصطناعي المحسنة.
-   **تخصيص مهارات التداول**: قم بتخصيص مهارات التداول لتناسب الاستراتيجيات الفردية.
-   **واجهة مستخدم متعددة اللغات**: يدعم التطبيق **42 لغة**، ويوفر الكشف التلقائي عن لغة الجهاز واختيار اللغة يدويًا في الإعدادات.
-   **النسخ الاحتياطي/الاستعادة**: قم بعمل نسخة احتياطية واستعادة بيانات التطبيق الخاصة بك بأمان.
-   **معلومات المطور**: الوصول إلى معلومات وأدوات خاصة بالمطورين.
-   **أدلة**: أدلة شاملة لمساعدة المستخدمين على التنقل واستخدام التطبيق بفعالية.

## المكدس التقني
-   **اسم المشروع**: KanutsananFX
-   **الإطار**: React Native + Expo + TypeScript
-   **التنقل**: Expo Router
-   **إدارة الحالة**: React Context API
-   **التخزين المحلي**: AsyncStorage
-   **المنصات**: iOS و Android
-   **دعم اللغة**: 42 لغة
-   **اللغة الاحتياطية الافتراضية**: التايلاندية

## التثبيت
للبدء باستخدام KanutsananFX، اتبع هذه الخطوات:

1.  **استنساخ المستودع (Clone the repository)**:
    ```bash
    git clone [repository_url]
    cd KanutsananFX
    ```
    *(ملاحظة: استبدل `[repository_url]` بعنوان المستودع الفعلي)*

2.  **تثبيت التبعيات (Install dependencies)**:
    ```bash
    npm install
    # أو
    pnpm install
    ```

## تشغيل التطبيق
لتشغيل التطبيق في وضع التطوير:

```bash
npx expo start
```
سيؤدي هذا إلى فتح أدوات Expo Dev Tools في متصفحك، مما يتيح لك تشغيل التطبيق على جهازك أو المحاكي.

## البناء لنظامي Android و iOS

### إصدارات Android
-   **ملف APK لمعاينة Android (Android Preview APK)**:
    ```bash
    npm run build:android:apk
    ```
-   **ملف AAB لإنتاج Android (Android Production AAB)**:
    ```bash
    npm run build:android:production
    ```

### إصدارات iOS
-   **إصدار معاينة iOS (iOS Preview Build)**:
    ```bash
    npm run build:ios
    ```
-   **إصدار محاكي iOS (iOS Simulator Build)**:
    ```bash
    npm run build:ios:simulator
    ```
-   **إصدار إنتاج iOS (iOS Production Build)**:
    ```bash
    npm run build:ios:production
    ```

## كيفية الاستخدام
1.  **بدء تشغيل التطبيق**: بعد التثبيت والتشغيل، سيتم الكشف عن لغة جهازك تلقائيًا. يمكنك تغيير اللغة يدويًا من خلال قسم الإعدادات في التطبيق.
2.  **التفاعل مع وكيل الذكاء الاصطناعي**: استخدم وكيل الدردشة بالذكاء الاصطناعي للحصول على تحليلات السوق، وتوقعات التداول، والإجابة على أسئلتك المتعلقة بالفوركس.
3.  **تنفيذ أوامر التداول**: استخدم الأوامر المحددة المتاحة في التطبيق لتنفيذ الصفقات بسرعة وفعالية.
4.  **تخصيص مهارات التداول**: قم بضبط مهارات الذكاء الاصطناعي لتتوافق مع استراتيجيات التداول الخاصة بك للحصول على أداء أمثل.
5.  **النسخ الاحتياطي والاستعادة**: قم بعمل نسخ احتياطية منتظمة لبياناتك واستعدها عند الحاجة لضمان سلامة معلوماتك.

## دعم متعدد اللغات
