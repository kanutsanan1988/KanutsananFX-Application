## දළ විශ්ලේෂණය

KanutsananFX යනු Expo / React Native භාවිතයෙන් iOS සහ Android යන දෙකටම සහය දක්වන නවීන විදේශ විනිමය වෙළඳාම් AI නියෝජිත යෙදුමකි. මෙම යෙදුම ඔබගේ වෙළඳාම් අත්දැකීම විප්ලවීය වෙනසක් කිරීමට සැලසුම් කර ඇත, එය බුද්ධිමත් AI හැකියාවන් සහ බහුභාෂා සහාය සමඟින් ලොව පුරා සිටින වෙළඳුන් සඳහා ප්‍රවේශ විය හැකිය.

    ```bash

    ```

```bash
npm run ios
```

## Android සහ iOS සඳහා ගොඩනැගීම

### Android

*   **Android පෙරදසුන් APK ගොඩනැගීම**: 

    ```bash
    npm run build:android:apk
    ```

*   **Android නිෂ්පාදන AAB ගොඩනැගීම**: 

    ```bash
    npm run build:android:production
    ```

### iOS

*   **iOS පෙරදසුන් ගොඩනැගීම**: 

    ```bash
    npm run build:ios
    ```

*   **iOS සිමියුලේටර් ගොඩනැගීම**: 

    ```bash
    npm run build:ios:simulator
    ```

*   **iOS නිෂ්පාදන ගොඩනැගීම**: 

    ```bash
    npm run build:ios:production
    ```
