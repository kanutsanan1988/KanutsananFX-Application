# KanutsananFX - फोरेक्स ट्रेडिङ एआई एजेन्ट एप्लिकेसन

## सिंहावलोकन

KanutsananFX एक अत्याधुनिक फोरेक्स ट्रेडिङ एआई एजेन्ट एप्लिकेसन हो जुन **Expo / React Native** प्रयोग गरी **आईओएस (iOS)** र **एन्ड्रोइड (Android)** दुवै प्लेटफर्मका लागि विकसित गरिएको छ। यो अनुप्रयोगले प्रयोगकर्ताहरूलाई एआई-संचालित ट्रेडिङ क्षमताहरू प्रदान गर्दछ, जसले फोरेक्स बजारमा स्मार्ट र कुशल निर्णयहरू लिन मद्दत गर्दछ।

## मुख्य विशेषताहरू

KanutsananFX ले धेरै मुख्य विशेषताहरू प्रदान गर्दछ जसले यसलाई फोरेक्स ट्रेडिङका लागि एक शक्तिशाली उपकरण बनाउँछ। यसमा ट्रेडिङ रणनीतिहरू र बजार विश्लेषणका लागि **एआई च्याट एजेन्ट (AI Chat Agent)** समावेश छ। प्रयोगकर्ताहरूले ट्रेडिङ कार्यहरू सजिलै सञ्चालन गर्नका लागि **४ ट्रेडिङ कमाण्डहरू (4 Trading Commands)** प्रयोग गर्न सक्छन्। **मेटाएपीआई एकीकरण (MetaAPI Integration)** ले वास्तविक-समय ट्रेडिङ र डाटा पहुँच सुनिश्चित गर्दछ, जबकि **बहु-एलएलएम समर्थन (Multi-LLM Support)** ले एआई क्षमताहरूमा लचिलोपन प्रदान गर्दछ। प्रयोगकर्ताहरूले आफ्नो आवश्यकता अनुसार **ट्रेडिङ स्किल अनुकूलन (Trading Skill Customization)** गर्न सक्छन्। अनुप्रयोगले **बहुभाषी यूआई (Multilingual UI)** मार्फत **४२ भाषाहरू** लाई समर्थन गर्दछ, जसमा यन्त्रको भाषा स्वतः पत्ता लगाउने र सेटिङहरूमा म्यानुअल भाषा चयन गर्ने विकल्प समावेश छ (पूर्वनिर्धारित भाषा थाई हो)। डाटा सुरक्षाका लागि **ब्याकअप/पुनर्स्थापना (Backup/Restore)** कार्यक्षमता उपलब्ध छ। यसका अतिरिक्त, **विकासकर्ता जानकारी (Developer Info)** र विस्तृत **गाइडहरू (Guides)** पनि प्रदान गरिएको छ।

## प्राविधिक स्ट्याक

KanutsananFX निम्न अत्याधुनिक प्रविधिहरू प्रयोग गरी निर्माण गरिएको छ: **फ्रेमवर्क (Framework)** को रूपमा React Native + Expo + TypeScript, **नेभिगेसन (Navigation)** का लागि Expo Router, **स्टेट व्यवस्थापन (State Management)** का लागि React Context API, **भण्डारण (Storage)** का लागि AsyncStorage, र **प्लेटफर्महरू (Platforms)** को रूपमा आईओएस (iOS) र एन्ड्रोइड (Android) दुवैलाई समर्थन गर्दछ।

## स्थापना

परियोजना सेट अप गर्नका लागि निम्न चरणहरू पालना गर्नुहोस्:

1.  रिपोजिटरी क्लोन गर्नुहोस्:

    ```bash
    git clone https://github.com/your-username/KanutsananFX.git
    cd KanutsananFX
    ```

2.  निर्भरताहरू स्थापना गर्नुहोस् (npm वा pnpm प्रयोग गरेर):

    ```bash
    npm install
    # वा
    pnpm install
    ```
## एप चलाउने

विकास सर्भर सुरु गर्नका लागि:

```bash
npx expo start
```

यसले एउटा QR कोड उत्पन्न गर्नेछ जुन तपाईंले Expo Go एप प्रयोग गरेर स्क्यान गर्न सक्नुहुन्छ (iOS र Android दुवैमा उपलब्ध)।

सिधै एन्ड्रोइड इमुलेटरमा चलाउनका लागि:

```bash
npm run android
```

सिधै आईओएस सिमुलेटरमा चलाउनका लागि:

```bash
npm run ios
```
## एन्ड्रोइड र आईओएसका लागि निर्माण

### एन्ड्रोइड

*   **एन्ड्रोइड पूर्वावलोकन APK निर्माण गर्न:**

    ```bash
    npm run build:android:apk
    ```

*   **एन्ड्रोइड उत्पादन AAB निर्माण गर्न:**

    ```bash
    npm run build:android:production
    ```

*   **आईओएस पूर्वावलोकन निर्माण गर्न:**

    ```bash
    npm run build:ios
    ```

*   **आईओएस सिमुलेटर निर्माण गर्न:**

    ```bash
    npm run build:ios:simulator
    ```

*   **आईओएस उत्पादन निर्माण गर्न:**

    ```bash
    npm run build:ios:production
    ```
## कसरी प्रयोग गर्ने

KanutsananFX एप्लिकेसन प्रयोग गर्नका लागि, यसलाई आफ्नो iOS वा Android उपकरणमा स्थापना गर्नुहोस्। एकपटक स्थापना भएपछि, तपाईंले एआई च्याट एजेन्टसँग अन्तर्क्रिया गर्न सक्नुहुन्छ, ट्रेडिङ कमाण्डहरू प्रयोग गर्न सक्नुहुन्छ, र MetaAPI एकीकरणको लाभ उठाउन सक्नुहुन्छ। सेटिङहरूमा, तपाईंले आफ्नो मनपर्ने भाषा चयन गर्न सक्नुहुन्छ वा यन्त्रको भाषा स्वतः पत्ता लगाउन दिन सक्नुहुन्छ। विस्तृत निर्देशनहरूको लागि, अनुप्रयोग भित्रका गाइडहरू हेर्नुहोस्।
