# KanutsananFX - फॉरेक्स ट्रेडिंग एआय एजंट ॲप्लिकेशन

KanutsananFX हे एक अत्याधुनिक फॉरेक्स ट्रेडिंग एआय एजंट ॲप्लिकेशन आहे जे **iOS आणि Android** दोन्ही प्लॅटफॉर्मवर **Expo / React Native** वापरून उपलब्ध आहे. हे ॲप्लिकेशन वापरकर्त्यांना कृत्रिम बुद्धिमत्तेच्या मदतीने फॉरेक्स ट्रेडिंगमध्ये सहभागी होण्यास सक्षम करते.

## विहंगावलोकन
KanutsananFX हे एक शक्तिशाली मोबाइल ॲप्लिकेशन आहे जे तुम्हाला फॉरेक्स मार्केटमध्ये एआय-आधारित ट्रेडिंगची शक्ती देते. हे ॲप **iOS आणि Android** दोन्ही उपकरणांवर अखंडपणे कार्य करते, कारण ते **Expo / React Native** फ्रेमवर्क वापरून तयार केले आहे. यात 42 भाषांसाठी समर्थन आहे, ज्यात डिव्हाइस-भाषेची स्वयंचलित ओळख आणि सेटिंग्जमध्ये मॅन्युअल भाषा निवडण्याची सुविधा समाविष्ट आहे.

## प्रमुख वैशिष्ट्ये

*   **एआय चॅट एजंट:** ट्रेडिंग धोरणे आणि मार्केट विश्लेषणासाठी बुद्धिमान एआय चॅट एजंटशी संवाद साधा.
*   **4 ट्रेडिंग कमांड्स:** खरेदी, विक्री, स्टॉप लॉस सेट करणे आणि टेक प्रॉफिट सेट करणे यासारख्या आवश्यक ट्रेडिंग कमांड्सचा वापर करा.
*   **MetaAPI एकीकरण:** MetaAPI द्वारे थेट ट्रेडिंग प्लॅटफॉर्मशी अखंडपणे कनेक्ट व्हा.
*   **मल्टी-एलएलएम समर्थन:** विविध लार्ज लँग्वेज मॉडेल्स (LLMs) वापरून वर्धित एआय क्षमतांचा अनुभव घ्या.
*   **ट्रेडिंग कौशल्य सानुकूलन:** तुमच्या ट्रेडिंग गरजांनुसार एआय एजंटची कौशल्ये सानुकूलित करा.
*   **बहुभाषिक UI:** ॲप्लिकेशन इंटरफेस 42 भाषांमध्ये उपलब्ध आहे, ज्यामुळे जागतिक वापरकर्त्यांसाठी सुलभता येते.
*   **बॅकअप आणि रिस्टोअर:** तुमच्या डेटाची सुरक्षितता सुनिश्चित करण्यासाठी बॅकअप आणि रिस्टोअर कार्यक्षमता.
*   **डेव्हलपर माहिती आणि मार्गदर्शक:** डेव्हलपर्ससाठी तपशीलवार माहिती आणि वापरकर्त्यांसाठी उपयुक्त मार्गदर्शक.

## तंत्रज्ञान स्टॅक

*   **प्रोजेक्ट नाव:** KanutsananFX
*   **फ्रेमवर्क:** React Native + Expo + TypeScript
*   **नेव्हिगेशन:** Expo Router
*   **स्टेट मॅनेजमेंट:** React Context API
*   **स्टोरेज:** AsyncStorage
*   **प्लॅटफॉर्म:** iOS आणि Android
*   **भाषा समर्थन:** 42 भाषा
*   **डीफॉल्ट फॉलबॅक भाषा:** थाई

## इन्स्टॉलेशन

प्रोजेक्ट सेट करण्यासाठी, खालील पायऱ्या फॉलो करा:

1.  रिपॉझिटरी क्लोन करा:
    ```bash
    git clone https://github.com/your-username/KanutsananFX.git
    cd KanutsananFX
    ```

2.  डिपेंडन्सीज इन्स्टॉल करा:
    ```bash
    npm install
    # किंवा
    pnpm install
    ```

## ॲप रन करणे

डेव्हलपमेंट सर्व्हर सुरू करण्यासाठी:

```bash
npx expo start
```

ॲप तुमच्या डिव्हाइसवर किंवा एमुलेटरवर रन करण्यासाठी:

*   **Android साठी:**
    ```bash
    npm run android
    ```

*   **iOS साठी:**
    ```bash
    npm run ios
    ```

## Android आणि iOS साठी बिल्ड करणे

### Android बिल्ड सूचना

*   **Android प्रीव्ह्यू APK बिल्ड करण्यासाठी:**
    ```bash
    npm run build:android:apk
    ```

*   **Android प्रोडक्शन AAB (ॲप बंडल) बिल्ड करण्यासाठी:**
    ```bash
    npm run build:android:production
    ```

### iOS बिल्ड सूचना

*   **iOS प्रीव्ह्यू बिल्ड करण्यासाठी:**
    ```bash
    npm run build:ios
    ```

*   **iOS सिम्युलेटर बिल्ड करण्यासाठी:**
    ```bash
    npm run build:ios:simulator
    ```

*   **iOS प्रोडक्शन बिल्ड करण्यासाठी:**
    ```bash
    npm run build:ios:production
    ```

ॲप सुरू केल्यानंतर, तुम्ही एआय चॅट एजंटशी संवाद साधू शकता, ट्रेडिंग कमांड्स वापरू शकता आणि तुमच्या ट्रेडिंग धोरणांना सानुकूलित करू शकता. सेटिंग्जमध्ये जाऊन तुम्ही ॲपची भाषा बदलू शकता.
