# KanutsananFX - Aplikace agenta AI pro obchodování na Forexu

## Přehled
KanutsananFX je sofistikovaná aplikace AI agenta pro obchodování na Forexu navržená tak, aby umožnila obchodníkům inteligentní automatizaci a přehledy trhu v reálném čase.Tato aplikace, postavená na **Expo / React Native**, zajišťuje bezproblémový výkon na platformách **iOS i Android** a poskytuje prostředí jako nativní na jakémkoli zařízení.

## Klíčové vlastnosti
* **AI Chat Agent**: Komunikujte s inteligentním asistentem AI pro obchodní poradenství a analýzu trhu.
* **4 obchodní příkazy**: Provádějte základní obchodní operace přímo prostřednictvím aplikace.
* **Integrace MetaAPI**: Propojuje se s MetaAPI pro tržní data a provádění obchodů v reálném čase.
* **Podpora více LLM**: Využívá několik velkých jazykových modelů pro vylepšené možnosti umělé inteligence.
* **Přizpůsobení obchodních dovedností**: Přizpůsobte obchodní strategie AI tak, aby vyhovovaly individuálním preferencím.
* **Vícejazyčné uživatelské rozhraní**: Podporuje 42 jazyků s automatickou detekcí jazyka zařízení a ručním výběrem v nastavení.
* **Zálohování/obnovení**: Bezpečně zálohujte a obnovte data aplikace.
* **Informace a příručky pro vývojáře**: Komplexní zdroje pro vývojáře a uživatele.

## Tech Stack
* **Název projektu**: KanutsananFX
* **Framework**: React Native + Expo + TypeScript
* **Navigace**: Expo Router
* **Správa státu**: React Context API
* **Úložiště**: AsyncStorage
* **Platformy**: iOS a Android
* **Jazyková podpora**: 42 jazyků
* **Výchozí záložní jazyk**: thajština

## Instalace
Chcete-li začít s KanutsananFX, postupujte takto:

1. **Klonujte úložiště**:
``` bash
git klon <url_repository>
cd KanutsananFX
```
*(Poznámka: Jakmile bude k dispozici, nahraďte „<url_repozitáře>“ skutečnou adresou URL úložiště.)*

2. **Instalovat závislosti**:
``` bash
instalace npm
# nebo
instalace pnpm
```
