# KanutsananFX - ফরেক্স ট্রেডিং এআই এজেন্ট অ্যাপ্লিকেশন

## সংক্ষিপ্ত বিবরণ
KanutsananFX হল একটি উন্নত ফরেক্স ট্রেডিং এআই এজেন্ট অ্যাপ্লিকেশন যা Expo এবং React Native ব্যবহার করে তৈরি করা হয়েছে, যা iOS এবং Android উভয় প্ল্যাটফর্মের জন্য নির্বিঘ্ন সমর্থন প্রদান করে। এই অ্যাপ্লিকেশনটি ট্রেডারদের একটি বুদ্ধিমান এআই চ্যাট এজেন্ট এবং শক্তিশালী ট্রেডিং কার্যকারিতা প্রদান করে, যা MetaAPI-এর সাথে একত্রিত। এটি বিশ্বব্যাপী ব্যবহারকারীদের জন্য ডিজাইন করা হয়েছে, স্বয়ংক্রিয় ডিভাইস ভাষা সনাক্তকরণ এবং ম্যানুয়াল ভাষা নির্বাচন বিকল্প সহ ৪২টি ভাষা সমর্থন করে।

## মূল বৈশিষ্ট্য
*   **এআই চ্যাট এজেন্ট:** ট্রেডিং অন্তর্দৃষ্টি এবং সহায়তার জন্য একটি বুদ্ধিমান এআই-এর সাথে যোগাযোগ করুন।
*   **৪টি ট্রেডিং কমান্ড:** অ্যাপের মাধ্যমে সরাসরি প্রয়োজনীয় ট্রেডিং অপারেশনগুলি সম্পাদন করুন।
*   **MetaAPI ইন্টিগ্রেশন:** রিয়েল-টাইম ট্রেডিং ডেটা এবং এক্সিকিউশনের জন্য MetaAPI-এর সাথে নির্বিঘ্নে সংযোগ করুন।
*   **মাল্টি-এলএলএম সমর্থন:** উন্নত এআই ক্ষমতার জন্য বিভিন্ন বৃহৎ ভাষা মডেল ব্যবহার করুন।
*   **ট্রেডিং দক্ষতা কাস্টমাইজেশন:** ট্রেডিং কৌশল এবং দক্ষতা ব্যক্তিগতকৃত এবং সূক্ষ্ম-টিউন করুন।
*   **বহুভাষিক ইউআই:** স্বয়ংক্রিয় সনাক্তকরণ এবং ম্যানুয়াল নির্বাচন সহ ৪২টি ভাষায় ব্যবহারকারী ইন্টারফেস উপলব্ধ।
*   **ব্যাকআপ/রিস্টোর:** আপনার অ্যাপ্লিকেশন ডেটা এবং সেটিংস নিরাপদে ব্যাকআপ এবং রিস্টোর করুন।
*   **ডেভেলপার তথ্য:** ডেভেলপারদের জন্য বিস্তারিত তথ্য অ্যাক্সেস করুন।
*   **গাইড:** অ্যাপ্লিকেশনটি কার্যকরভাবে ব্যবহারের জন্য ব্যাপক গাইড।

## প্রযুক্তিগত স্ট্যাক
*   **ফ্রেমওয়ার্ক:** React Native + Expo + TypeScript
*   **নেভিগেশন:** Expo Router
*   **স্টেট ম্যানেজমেন্ট:** React Context API
*   **স্থানীয় স্টোরেজ:** AsyncStorage
*   **প্ল্যাটফর্ম:** iOS এবং Android

## ইনস্টলেশন
ডেভেলপমেন্ট এনভায়রনমেন্ট সেট আপ করতে, এই পদক্ষেপগুলি অনুসরণ করুন:

১.  **রিপোজিটরি ক্লোন করুন:**
    ```bash
    git clone [repository_url]
    cd KanutsananFX
    ```

২.  **ডিপেন্ডেন্সি ইনস্টল করুন:**
    ```bash
    npm install
    # অথবা
    pnpm install
    ```

## অ্যাপ চালানো
ডেভেলপমেন্ট মোডে অ্যাপ্লিকেশনটি চালাতে:

```bash
npx expo start
```
এটি আপনার ব্রাউজারে Expo Dev Tools খুলবে। আপনি একটি ফিজিক্যাল ডিভাইস বা এমুলেটরে অ্যাপটি চালানোর জন্য নির্দেশাবলী অনুসরণ করতে পারেন।

## Android এবং iOS এর জন্য বিল্ড করা

### Android বিল্ড নির্দেশাবলী

*   **Android প্রিভিউ APK বিল্ড করুন:**
    ```bash
    npm run build:android:apk
    ```
    এটি একটি `.apk` ফাইল তৈরি করবে যা Android ডিভাইসে ইনস্টল করা যেতে পারে পরীক্ষার উদ্দেশ্যে।

*   **Android প্রোডাকশন AAB বিল্ড করুন:**
    ```bash
    npm run build:android:production
    ```
    এটি একটি `.aab` (Android App Bundle) ফাইল তৈরি করবে যা Google Play Store-এ আপলোড করার জন্য উপযুক্ত।

### iOS বিল্ড নির্দেশাবলী

*   **iOS প্রিভিউ বিল্ড করুন:**
    ```bash
    npm run build:ios
    ```
    এটি Expo Go অ্যাপ ব্যবহার করে একটি প্রিভিউ বিল্ড তৈরি করবে।

*   **iOS সিমুলেটর বিল্ড করুন:**
    ```bash
    npm run build:ios:simulator
    ```
    এটি একটি `.tar.gz` ফাইল তৈরি করবে যা Xcode সিমুলেটরে ইনস্টল করা যেতে পারে।

*   **iOS প্রোডাকশন বিল্ড করুন:**
    ```bash
    npm run build:ios:production
    ```
    এটি একটি `.ipa` ফাইল তৈরি করবে যা Apple App Store-এ আপলোড করার জন্য উপযুক্ত।

## কিভাবে ব্যবহার করবেন
অ্যাপ্লিকেশনটি ব্যবহার করার জন্য, নিম্নলিখিত নির্দেশাবলী অনুসরণ করুন:

১.  **লগইন/সাইনআপ:** আপনার ট্রেডিং অ্যাকাউন্ট অ্যাক্সেস করতে লগইন করুন বা একটি নতুন অ্যাকাউন্ট তৈরি করুন।
২.  **MetaAPI সংযোগ:** আপনার MetaAPI অ্যাকাউন্ট সংযুক্ত করুন যাতে অ্যাপটি আপনার ট্রেডিং ডেটা অ্যাক্সেস করতে পারে এবং ট্রেড সম্পাদন করতে পারে।
৩.  **এআই চ্যাট:** ট্রেডিং অন্তর্দৃষ্টি, বাজার বিশ্লেষণ বা সাধারণ প্রশ্নের জন্য এআই চ্যাট এজেন্টের সাথে যোগাযোগ করুন।
৪.  **ট্রেডিং কমান্ড:** অ্যাপের মধ্যে উপলব্ধ ৪টি ট্রেডিং কমান্ড ব্যবহার করে ট্রেড করুন।
৫.  **সেটিংস:** আপনার ট্রেডিং দক্ষতা কাস্টমাইজ করতে, ভাষা পরিবর্তন করতে বা ডেটা ব্যাকআপ/রিস্টোর করতে সেটিংস মেনু অন্বেষণ করুন।

## বহুভাষিক সমর্থন
KanutsananFX অ্যাপ্লিকেশনটি ৪২টি ভাষা সমর্থন করে, যা বিশ্বব্যাপী ব্যবহারকারীদের জন্য একটি অন্তর্ভুক্তিমূলক অভিজ্ঞতা নিশ্চিত করে। অ্যাপ্লিকেশনটি স্বয়ংক্রিয়ভাবে আপনার ডিভাইসের ভাষা সনাক্ত করে এবং সেই অনুযায়ী ইউআই সামঞ্জস্য করে। উপরন্তু, আপনি সেটিংস মেনুতে ম্যানুয়ালি আপনার পছন্দের ভাষা নির্বাচন করতে পারেন। ডিফল্ট ফলব্যাক ভাষা হল থাই।
