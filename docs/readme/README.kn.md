# KanutsananFX - ವಿದೇಶೀ ವಿನಿಮಯ ವ್ಯಾಪಾರ AI ಏಜೆಂಟ್ ಅಪ್ಲಿಕೇಶನ್

## ಅವಲೋಕನ

KanutsananFX ಸುಧಾರಿತ ಫಾರೆಕ್ಸ್ ಟ್ರೇಡಿಂಗ್ AI ಏಜೆಂಟ್ ಅಪ್ಲಿಕೇಶನ್ ಆಗಿದ್ದು, **iOS ಮತ್ತು Android** ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ಗಳಿಗಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ, ಇದನ್ನು **Expo / React Native** ಬಳಸಿ ನಿರ್ಮಿಸಲಾಗಿದೆ.ಈ ಅಪ್ಲಿಕೇಶನ್ ಬುದ್ಧಿವಂತ ವ್ಯಾಪಾರ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಹೆಚ್ಚು ಗ್ರಾಹಕೀಯಗೊಳಿಸಬಹುದಾದ, ಬಹುಭಾಷಾ ಬಳಕೆದಾರ ಇಂಟರ್ಫೇಸ್ನೊಂದಿಗೆ ಬಳಕೆದಾರರಿಗೆ ಅಧಿಕಾರ ನೀಡುತ್ತದೆ.

## ಪ್ರಮುಖ ಲಕ್ಷಣಗಳು

*   **AI ಚಾಟ್ ಏಜೆಂಟ್**: ಟ್ರೇಡಿಂಗ್ ಒಳನೋಟಗಳು ಮತ್ತು ಸಹಾಯಕ್ಕಾಗಿ ಬುದ್ಧಿವಂತ AI ಏಜೆಂಟ್‌ನೊಂದಿಗೆ ಸಂವಹನ ನಡೆಸಿ.
*   **4 ಟ್ರೇಡಿಂಗ್ ಕಮಾಂಡ್‌ಗಳು**: ಅಪ್ಲಿಕೇಶನ್ ಮೂಲಕ ನೇರವಾಗಿ ಅಗತ್ಯ ವ್ಯಾಪಾರ ಕಾರ್ಯಾಚರಣೆಗಳನ್ನು ಕಾರ್ಯಗತಗೊಳಿಸಿ.
*   **MetaAPI ಇಂಟಿಗ್ರೇಷನ್**: ನೈಜ-ಸಮಯದ ಮಾರುಕಟ್ಟೆ ಡೇಟಾ ಮತ್ತು ವ್ಯಾಪಾರ ಕಾರ್ಯಗತಗೊಳಿಸಲು MetaAPI ನೊಂದಿಗೆ ಮನಬಂದಂತೆ ಸಂಪರ್ಕಪಡಿಸಿ.
*   **ಮಲ್ಟಿ-ಎಲ್‌ಎಲ್‌ಎಂ ಬೆಂಬಲ**: ವರ್ಧಿತ AI ಸಾಮರ್ಥ್ಯಗಳಿಗಾಗಿ ವಿವಿಧ ದೊಡ್ಡ ಭಾಷಾ ಮಾದರಿಗಳನ್ನು ನಿಯಂತ್ರಿಸಿ.
*   **ಟ್ರೇಡಿಂಗ್ ಸ್ಕಿಲ್ ಕಸ್ಟಮೈಸೇಶನ್**: ವೈಯಕ್ತಿಕ ಆದ್ಯತೆಗಳಿಗೆ ಸರಿಹೊಂದುವಂತೆ ವ್ಯಾಪಾರ ತಂತ್ರಗಳನ್ನು ವೈಯಕ್ತೀಕರಿಸಿ ಮತ್ತು ಪರಿಷ್ಕರಿಸಿ.
*   **ಬಹುಭಾಷಾ UI**: ಸ್ವಯಂಚಾಲಿತ ಸಾಧನ ಭಾಷೆ ಪತ್ತೆ ಮತ್ತು ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಹಸ್ತಚಾಲಿತ ಆಯ್ಕೆಯ ಆಯ್ಕೆಗಳೊಂದಿಗೆ **42 ಭಾಷೆಗಳಲ್ಲಿ** ಲಭ್ಯವಿರುವ ಬಳಕೆದಾರ ಇಂಟರ್ಫೇಸ್ ಅನ್ನು ಆನಂದಿಸಿ.
*   **ಬ್ಯಾಕಪ್/ರೀಸ್ಟೋರ್**: ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಡೇಟಾವನ್ನು ಸುರಕ್ಷಿತವಾಗಿ ಬ್ಯಾಕಪ್ ಮಾಡಿ ಮತ್ತು ಮರುಸ್ಥಾಪಿಸಿ.
*   **ಡೆವಲಪರ್ ಮಾಹಿತಿ**: ಡೆವಲಪರ್‌ಗಳಿಗಾಗಿ ವಿವರವಾದ ಮಾಹಿತಿಯನ್ನು ಪ್ರವೇಶಿಸಿ.
*   **ಮಾರ್ಗದರ್ಶಿಗಳು**: ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಲು ಮತ್ತು ಕರಗತ ಮಾಡಿಕೊಳ್ಳಲು ನಿಮಗೆ ಸಹಾಯ ಮಾಡಲು ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿಗಳು.

## ಟೆಕ್ ಸ್ಟಾಕ್

*   **ಫ್ರೇಮ್‌ವರ್ಕ್**: ರಿಯಾಕ್ಟ್ ಸ್ಥಳೀಯ + ಎಕ್ಸ್‌ಪೋ + ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್
*   **ನ್ಯಾವಿಗೇಷನ್**: ಎಕ್ಸ್‌ಪೋ ರೂಟರ್
*   **ರಾಜ್ಯ ನಿರ್ವಹಣೆ**: ಪ್ರತಿಕ್ರಿಯೆ ಸಂದರ್ಭ API
*   **ಸ್ಥಳೀಯ ಸಂಗ್ರಹಣೆ**: AsyncStorage
*   **ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ಗಳು**: iOS ಮತ್ತು Android

## ಅನುಸ್ಥಾಪನೆ

ಅಭಿವೃದ್ಧಿ ಪರಿಸರವನ್ನು ಹೊಂದಿಸಲು, ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ:

1.  **ರೆಪೊಸಿಟರಿಯನ್ನು ಕ್ಲೋನ್ ಮಾಡಿ**:

    ```bash
    git clone https://github.com/kanutsanan1988/KanutsananFX.git
    cd KanutsananFX
    ```

2.  **ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ**:

    ```bash
    npm install
    # ಅಥವಾ
    pnpm install
    ```

## ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ರನ್ ಮಾಡಲಾಗುತ್ತಿದೆ

ಡೆವಲಪ್‌ಮೆಂಟ್ ಮೋಡ್‌ನಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಚಲಾಯಿಸಲು:

```bash
npx expo start
```

ಇದು ನಿಮ್ಮ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ಎಕ್ಸ್‌ಪೋ ಡೆವಲಪರ್ ಪರಿಕರಗಳನ್ನು ತೆರೆಯುತ್ತದೆ.ನಂತರ ನೀವು ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಚಲಾಯಿಸಲು ಆಯ್ಕೆ ಮಾಡಬಹುದು:

*   Android ಎಮ್ಯುಲೇಟರ್ ಅಥವಾ ಸಾಧನ (`npm run android`)
*   iOS ಸಿಮ್ಯುಲೇಟರ್ ಅಥವಾ ಸಾಧನ (`npm run ios`)

## Android ಮತ್ತು iOS ಗಾಗಿ ಕಟ್ಟಡ

### Android ಬಿಲ್ಡ್‌ಗಳು

*   **ಆಂಡ್ರಾಯ್ಡ್ ಪೂರ್ವವೀಕ್ಷಣೆ APK**:

    ```bash
    npm run build:android:apk
    ```

*   **ಆಂಡ್ರಾಯ್ಡ್ ಪ್ರೊಡಕ್ಷನ್ AAB**:

    ```bash
    npm run build:android:production
    ```

### iOS ಬಿಲ್ಡ್‌ಗಳು

*   **iOS ಪೂರ್ವವೀಕ್ಷಣೆ ಬಿಲ್ಡ್**:

    ```bash
    npm run build:ios
    ```

*   **iOS ಸಿಮ್ಯುಲೇಟರ್ ಬಿಲ್ಡ್**:

    ```bash
    npm run build:ios:simulator
    ```

*   **iOS ಪ್ರೊಡಕ್ಷನ್ ಬಿಲ್ಡ್**:

    ```bash
    npm run build:ios:production
    ```
