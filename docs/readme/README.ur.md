# KanutsananFX - Forex Trading AI Agent Application

## جائزہ (Overview)
KanutsananFX ایک جدید فاریکس ٹریڈنگ AI ایجنٹ ایپلیکیشن ہے جو iOS اور Android دونوں پلیٹ فارمز پر Expo / React Native کا استعمال کرتے ہوئے دستیاب ہے۔ یہ ایپلیکیشن خودکار ٹریڈنگ کے لیے ڈیزائن کی گئی ہے، جس میں AI چیٹ ایجنٹ، MetaAPI انٹیگریشن، اور متعدد LLM سپورٹ شامل ہیں۔

## اہم خصوصیات (Key Features)
*   **AI چیٹ ایجنٹ:** ذہین گفتگو کے ذریعے ٹریڈنگ کے فیصلے کرنے میں مدد کرتا ہے۔
*   **4 ٹریڈنگ کمانڈز:** مخصوص ٹریڈنگ آپریشنز کے لیے چار بلٹ ان کمانڈز۔
*   **MetaAPI انٹیگریشن:** MetaTrader پلیٹ فارمز کے ساتھ ہموار انضمام۔
*   **متعدد LLM سپورٹ:** مختلف بڑے لسانی ماڈلز (LLMs) کے ساتھ مطابقت۔
*   **ٹریڈنگ سکل کسٹمائزیشن:** اپنی ٹریڈنگ کی مہارتوں کو اپنی مرضی کے مطابق بنائیں۔
*   **کثیر لسانی UI:** 42 زبانوں کی حمایت کرتا ہے، جس میں ڈیوائس کی زبان کا خودکار پتہ لگانا اور سیٹنگز میں دستی زبان کا انتخاب شامل ہے۔
*   **بیک اپ/ریسٹور:** آسانی سے ڈیٹا کا بیک اپ لیں اور بحال کریں۔
*   **ڈویلپر معلومات اور گائیڈز:** ڈویلپرز کے لیے تفصیلی معلومات اور استعمال کے گائیڈز۔

## ٹیکنالوجی اسٹیک (Tech Stack)
*   **پروجیکٹ کا نام:** KanutsananFX
*   **فریم ورک:** React Native + Expo + TypeScript
*   **نیویگیشن:** Expo Router
*   **اسٹیٹ مینجمنٹ:** React Context API
*   **اسٹوریج:** AsyncStorage
*   **پلیٹ فارمز:** iOS اور Android
*   **زبان کی حمایت:** 42 زبانیں (ڈیفالٹ فال بیک زبان: تھائی)

## انسٹالیشن (Installation)
پروجیکٹ کو سیٹ اپ کرنے کے لیے ان مراحل پر عمل کریں:

```bash
npm install
# یا
pnpm install
```

## ایپ چلانا (Running the App)
ڈویلپمنٹ سرور شروع کرنے کے لیے:

```bash
npx expo start
```

Android ایمولیٹر پر چلانے کے لیے:

```bash
npm run android
```

iOS ایمولیٹر پر چلانے کے لیے:

```bash
npm run ios
```

## Android اور iOS کے لیے بلڈنگ (Building for Android and iOS)

### Android

**پریویو APK بنانا:**

```bash
npm run build:android:apk
```

**پروڈکشن AAB بنانا:**

```bash
npm run build:android:production
```

### iOS

**پریویو بلڈ بنانا:**

```bash
npm run build:ios
```

**iOS سمیلیٹر بلڈ بنانا:**

```bash
npm run build:ios:simulator
```

**پروڈکشن بلڈ بنانا:**

```bash
npm run build:ios:production
```

## استعمال کا طریقہ (How to Use)
ایپلیکیشن شروع کرنے کے بعد، آپ AI چیٹ ایجنٹ کے ساتھ بات چیت کر سکتے ہیں یا MetaAPI انٹیگریشن کے ذریعے ٹریڈنگ کمانڈز استعمال کر سکتے ہیں۔ سیٹنگز میں اپنی پسندیدہ زبان منتخب کریں یا ڈیوائس کی زبان کا خودکار پتہ لگانے دیں۔

## کثیر لسانی سپورٹ (Multilingual Support)
KanutsananFX 42 زبانوں کو سپورٹ کرتا ہے، جس میں ڈیوائس کی زبان کا خودکار پتہ لگانا اور سیٹنگز میں دستی زبان کا انتخاب شامل ہے۔ ڈیفالٹ فال بیک زبان تھائی ہے۔
