# KanutsananFX - Додаток AI-агента для торгівлі на Форекс

## Огляд

KanutsananFX — це передовий додаток AI-агента для торгівлі на Форекс, розроблений для iOS та Android за допомогою **Expo / React Native**. Він надає користувачам потужні інструменти та можливості штучного інтелекту для ефективного управління торгівлею на Форекс. Додаток підтримує **42 мови** з автоматичним визначенням мови пристрою та можливістю ручного вибору мови в налаштуваннях.

## Ключові особливості

*   **AI Chat Agent**: Інтелектуальний чат-агент, який допомагає користувачам з аналізом ринку та торговими стратегіями.
*   **4 торгові команди**: Виконуйте ключові торгові операції безпосередньо з програми.
*   **Інтеграція з MetaAPI**: Безшовна інтеграція з MetaAPI для доступу до ринкових даних та виконання угод.
*   **Підтримка кількох LLM**: Використовуйте різні великі мовні моделі для розширених можливостей ШІ.
*   **Налаштування торгових навичок**: Персоналізуйте торгові навички агента відповідно до ваших уподобань.
*   **Багатомовний інтерфейс**: Зручний інтерфейс, доступний 42 мовами.
*   **Резервне копіювання та відновлення**: Забезпечте безпеку ваших даних за допомогою функцій резервного копіювання та відновлення.
*   **Інформація для розробників та посібники**: Детальна документація та посібники для розробників та просунутих користувачів.

## Технологічний стек

*   **Фреймворк**: React Native + Expo + TypeScript
*   **Навігація**: Expo Router
*   **Управління станом**: React Context API
*   **Зберігання даних**: AsyncStorage
*   **Платформи**: iOS та Android

## Встановлення

Щоб розпочати роботу з KanutsananFX, виконайте наступні кроки:

```bash
npm install
# або
pnpm install
```

## Запуск програми

Щоб запустити програму в режимі розробки:

```bash
npx expo start
```

Для запуску на Android-емуляторі/пристрої:

```bash
npm run android
```

Для запуску на iOS-симуляторі/пристрої:

```bash
npm run ios
```

## Збірка для Android та iOS

### Android

**Збірка попереднього APK:**

```bash
npm run build:android:apk
```

**Збірка виробничого AAB:**

```bash
npm run build:android:production
```

### iOS

**Збірка попередньої версії для iOS:**

```bash
npm run build:ios
```

**Збірка для симулятора iOS:**

```bash
npm run build:ios:simulator
```

**Збірка виробничої версії для iOS:**

```bash
npm run build:ios:production
```

## Як користуватися

(Тут будуть детальні інструкції щодо використання програми після її встановлення та запуску. Наприклад, як налаштувати торгові команди, підключитися до MetaAPI, використовувати AI Chat Agent тощо.)

## Багатомовна підтримка

KanutsananFX підтримує 42 мови, забезпечуючи глобальний доступ. Додаток автоматично визначає мову вашого пристрою, але ви також можете вручну вибрати бажану мову в розділі "Налаштування". Мова за замовчуванням (fallback) — тайська.
