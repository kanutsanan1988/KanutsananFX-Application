# KanutsananFX - יישום סוכן AI למסחר במט"ח

## סקירה כללית

KanutsananFX הוא יישום סוכן מסחר במט"ח מתקדם המופעל על ידי AI, שנועד להעצים סוחרים באמצעות אוטומציה חכמה ותובנות שוק בזמן אמת. נבנה עם **React Native** ו-**Expo**, יישום זה מספק תמיכה חלקה בין פלטפורמות עבור מכשירי **iOS ו-Android**, ומבטיח חווית מסחר עקבית וחזקה בפלטפורמה הניידת המועדפת עליכם.

עם התמקדות בנגישות והישג יד גלובלי, KanutsananFX תומך כעת ב-**42 שפות**, וכולל זיהוי אוטומטי של שפת המכשיר ואת הגמישות למשתמשים לבחור ידנית את השפה המועדפת עליהם בהגדרות היישום. תמיכה רב-לשונית נרחבת זו מבטיחה שסוחרים ברחבי העולם יוכלו לנווט ולהשתמש ביישום בקלות ובביטחון.

## תכונות עיקריות

*   **סוכן צ\'אט AI**: אינטראקציה עם עוזר AI חכם לקבלת ייעוץ מסחרי, ניתוח שוק וגיבוש אסטרטגיה.
*   **4 פקודות מסחר**: ביצוע פעולות מסחר חיוניות ישירות דרך סוכן ה-AI, כולל פתיחה, סגירה, שינוי וניטור עסקאות.
*   **שילוב MetaAPI**: התחברות חלקה לחשבונות MetaTrader באמצעות MetaAPI לקבלת נתונים בזמן אמת, ביצוע עסקאות וניהול חשבונות.
*   **תמיכה בריבוי LLM**: מינוף מודלי שפה גדולים שונים (LLM) לשיפור יכולות סוכן ה-AI והתאמתו.
*   **התאמה אישית של מיומנויות מסחר**: התאמה אישית וכוונון עדין של אסטרטגיות מסחר והתנהגויות סוכן AI כדי להתאים לסגנונות והעדפות מסחר אישיים.
*   **ממשק משתמש רב-לשוני**: ליהנות מממשק משתמש מתורגם במלואו ב-42 שפות, עם אפשרויות זיהוי אוטומטי ובחירה ידנית.
*   **גיבוי/שחזור**: גיבוי ושחזור מאובטחים של נתוני היישום וההגדרות שלך.
*   **מידע למפתחים ומדריכים**: גישה למשאבים מקיפים למפתחים ומדריכים מפורטים למשתמשים כדי למקסם את תועלת היישום.

## ערימת טכנולוגיות

*   **Framework**: React Native + Expo + TypeScript
*   **ניווט**: Expo Router
*   **ניהול מצב**: React Context API
*   **אחסון מקומי**: AsyncStorage
*   **פלטפורמות**: iOS ו-Android
*   **תמיכה בשפות**: 42 שפות (שפת חלופה ברירת מחדל: תאילנדית)

## התקנה

כדי להגדיר את סביבת הפיתוח של KanutsananFX, בצע את השלבים הבאים:

1.  **שכפול המאגר**:

    ```bash
    git clone https://github.com/kanutsanan1988/KanutsananFX.git
    cd KanutsananFX
    ```

2.  **התקנת תלויות**:

    ```bash
    npm install
    # או
    pnpm install
    ```

## הפעלת היישום

כדי להפעיל את היישום במצב פיתוח:

```bash
npx expo start
```

פעולה זו תפעיל את שרת הפיתוח של Expo. לאחר מכן תוכל לפתוח את היישום במכשיר או בסימולטור שלך באמצעות יישום Expo Go.

## בנייה עבור Android ו-iOS

KanutsananFX תומך בתצורות בנייה שונות עבור פלטפורמות Android ו-iOS כאחד.

### בניית Android

*   **קובץ APK לתצוגה מקדימה של Android**:

    ```bash
    npm run build:android:apk
    ```

*   **קובץ AAB לייצור Android**:

    ```bash
    npm run build:android:production
    ```

### בניית iOS

*   **בניית תצוגה מקדימה של iOS**:

    ```bash
    npm run build:ios
    ```

*   **בניית סימולטור iOS**:

    ```bash
    npm run build:ios:simulator
    ```

*   **בניית ייצור iOS**:

    ```bash
    npm run build:ios:production
    ```
