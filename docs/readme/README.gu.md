# KanutsananFX - ફોરેક્સ ટ્રેડિંગ AI એજન્ટ એપ્લિકેશન

## વિહંગાવલોકન

KanutsananFX એ એક અત્યાધુનિક ફોરેક્સ ટ્રેડિંગ AI એજન્ટ એપ્લિકેશન છે જે iOS અને Android બંને પ્લેટફોર્મ પર **એક્સ્પો / રીએક્ટ નેટિવ** નો ઉપયોગ કરીને ઉપલબ્ધ છે. આ એપ્લિકેશન હવે **42 ભાષાઓ** ને સપોર્ટ કરે છે, જેમાં ઉપકરણ-ભાષા સ્વતઃ-શોધ અને સેટિંગ્સમાં મેન્યુઅલ ભાષા પસંદગીનો સમાવેશ થાય છે. તેનો હેતુ વપરાશકર્તાઓને ફોરેક્સ ટ્રેડિંગમાં મદદ કરવા માટે એક શક્તિશાળી અને બહુભાષી પ્લેટફોર્મ પ્રદાન કરવાનો છે.

## મુખ્ય લક્ષણો

*   **AI ચેટ એજન્ટ**: બુદ્ધિશાળી AI ચેટ એજન્ટ સાથે સીધો સંપર્ક કરો જે તમને ટ્રેડિંગ નિર્ણયો લેવામાં મદદ કરે છે.
*   **4 ટ્રેડિંગ આદેશો**: ખરીદી, વેચાણ, સ્ટોપ લોસ અને ટેક પ્રોફિટ સહિત 4 મુખ્ય ટ્રેડિંગ આદેશોનો ઉપયોગ કરો.
*   **MetaAPI એકીકરણ**: MetaAPI સાથે સીમલેસ એકીકરણ દ્વારા રીઅલ-ટાઇમ માર્કેટ ડેટા અને એક્ઝેક્યુશનનો લાભ લો.
*   **મલ્ટિ-એલએલએમ સપોર્ટ**: બહુવિધ મોટા ભાષા મોડેલો (LLMs) માટે સપોર્ટ સાથે લવચીકતા અને પ્રદર્શનમાં સુધારો.
*   **ટ્રેડિંગ કૌશલ્ય કસ્ટમાઇઝેશન**: તમારી જરૂરિયાતો અનુસાર ટ્રેડિંગ કૌશલ્યોને કસ્ટમાઇઝ કરો.
*   **બહુભાષી UI**: 42 ભાષાઓમાં ઉપલબ્ધ યુઝર ઇન્ટરફેસ સાથે વૈશ્વિક પહોંચ.
*   **બેકઅપ/રીસ્ટોર**: સરળ બેકઅપ અને રીસ્ટોર કાર્યો સાથે તમારા ડેટાને સુરક્ષિત રાખો.
*   **વિકાસકર્તા માહિતી અને માર્ગદર્શિકાઓ**: વિકાસકર્તાઓ માટે વિગતવાર માહિતી અને ઉપયોગી માર્ગદર્શિકાઓ ઉપલબ્ધ છે.

## ટેક સ્ટેક

*   **પ્રોજેક્ટ નામ**: KanutsananFX
*   **ફ્રેમવર્ક**: React Native + Expo + TypeScript
*   **નેવિગેશન**: Expo Router
*   **સ્ટેટ મેનેજમેન્ટ**: React Context API
*   **સ્ટોરેજ**: AsyncStorage
*   **પ્લેટફોર્મ્સ**: iOS અને Android
*   **ભાષા સપોર્ટ**: 42 ભાષાઓ
*   **ડિફોલ્ટ ફોલબેક ભાષા**: થાઈ

## સ્થાપન

આ પ્રોજેક્ટને સ્થાનિક રીતે સેટ કરવા માટે, નીચેના પગલાં અનુસરો:

1.  રિપોઝીટરી ક્લોન કરો:

    ```bash
    git clone https://github.com/kanutsanan1988/Forex-Auto-Trading-MetaAPI-Skill-for-OpenClaw-by-Kanutsanan-Pongpanna.git
    cd Forex-Auto-Trading-MetaAPI-Skill-for-OpenClaw-by-Kanutsanan-Pongpanna
    ```

2.  ડિપેન્ડન્સીઝ ઇન્સ્ટોલ કરો:

    ```bash
    npm install
    # અથવા
    pnpm install
    ```

3.  એપ્લિકેશન શરૂ કરો:

    ```bash
    npx expo start
    ```

    આ કમાન્ડ એક ડેવલપમેન્ટ સર્વર શરૂ કરશે અને QR કોડ પ્રદર્શિત કરશે. તમે Expo Go એપ્લિકેશનનો ઉપયોગ કરીને તમારા ઉપકરણ પર QR કોડ સ્કેન કરીને એપ્લિકેશન ચલાવી શકો છો.

## એપ ચલાવી રહ્યા છીએ

તમે નીચેના આદેશોનો ઉપયોગ કરીને સીધા જ તમારા ઉપકરણ પર એપ્લિકેશન ચલાવી શકો છો:

*   **Android પર ચલાવો**:

    ```bash
    npm run android
    ```

*   **iOS પર ચલાવો**:

    ```bash
    npm run ios
    ```

## Android અને iOS માટે બિલ્ડીંગ

એપ્લિકેશનના વિવિધ બિલ્ડ્સ બનાવવા માટે નીચેના આદેશોનો ઉપયોગ કરો:

*   **Android પ્રીવ્યુ APK**:

    ```bash
    npm run build:android:apk
    ```

*   **Android પ્રોડક્શન AAB**:

    ```bash
    npm run build:android:production
    ```

*   **iOS પ્રીવ્યુ બિલ્ડ**:

    ```bash
    npm run build:ios
    ```

*   **iOS સિમ્યુલેટર બિલ્ડ**:

    ```bash
    npm run build:ios:simulator
    ```

*   **iOS પ્રોડક્શન બિલ્ડ**:

    ```bash
    npm run build:ios:production
    ```

## કેવી રીતે ઉપયોગ કરવો

એપ્લિકેશનનો ઉપયોગ કરવા માટે, કૃપા કરીને નીચેના પગલાં અનુસરો:

1.  **AI ચેટ એજન્ટ**: AI ચેટ એજન્ટ સાથે વાતચીત કરીને ટ્રેડિંગ સલાહ મેળવો.
2.  **ટ્રેડિંગ આદેશો**: 4 મુખ્ય ટ્રેડિંગ આદેશોનો ઉપયોગ કરીને ટ્રેડ કરો.
3.  **ભાષા પસંદગી**: સેટિંગ્સમાં તમારી પસંદગીની ભાષા પસંદ કરો અથવા ઉપકરણ-ભાષા સ્વતઃ-શોધનો ઉપયોગ કરો.

## બહુભાષી આધાર

KanutsananFX એપ્લિકેશન 42 ભાષાઓને સપોર્ટ કરે છે, જે વપરાશકર્તાઓને તેમની પસંદગીની ભાષામાં એપ્લિકેશનનો ઉપયોગ કરવાની સુવિધા આપે છે. એપ્લિકેશન ઉપકરણની ભાષાને આપમેળે શોધી કાઢે છે અને તે મુજબ UI પ્રદર્શિત કરે છે. જો જરૂરી હોય, તો વપરાશકર્તાઓ સેટિંગ્સ મેનૂમાંથી મેન્યુઅલી ભાષા પસંદ કરી શકે છે. ડિફોલ્ટ ફોલબેક ભાષા થાઈ છે.
