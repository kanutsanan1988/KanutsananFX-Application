# कनुत्सनानएफएक्स - फॉरेक्स ट्रेडिंग एआई एजेंट एप्लीकेशन

## अवलोकन

कनुत्सनानएफएक्स एक उन्नत विदेशी मुद्रा व्यापार एआई एजेंट एप्लिकेशन है जिसे व्यापारियों को बुद्धिमान स्वचालन और व्यापक बाजार विश्लेषण के साथ सशक्त बनाने के लिए डिज़ाइन किया गया है।**रिएक्ट नेटिव** और **एक्सपो** के साथ निर्मित, यह एप्लिकेशन **आईओएस और एंड्रॉइड** दोनों डिवाइसों के लिए निर्बाध क्रॉस-प्लेटफ़ॉर्म समर्थन प्रदान करता है।यह ट्रेडिंग कमांड निष्पादित करने, मेटाएपीआई के साथ एकीकृत करने और ट्रेडिंग कौशल के लिए व्यापक अनुकूलन प्रदान करने के लिए अत्याधुनिक एआई क्षमताओं का लाभ उठाता है।डिवाइस-भाषा ऑटो-डिटेक्शन और मैन्युअल चयन सहित **42 भाषाओं** के समर्थन के साथ, कनुत्सनानएफएक्स वास्तव में वैश्विक और सुलभ ट्रेडिंग अनुभव सुनिश्चित करता है।

## मुख्य विशेषताएं

- **एआई चैट एजेंट**: ट्रेडिंग अंतर्दृष्टि और कमांड निष्पादन के लिए एक बुद्धिमान एआई सहायक के साथ बातचीत करें।
- **4 ट्रेडिंग कमांड**: सीधे एप्लिकेशन के माध्यम से आवश्यक ट्रेडिंग संचालन निष्पादित करें।
- **मेटाएपीआई एकीकरण**: वास्तविक समय बाजार डेटा और व्यापार निष्पादन के लिए मेटाएपीआई से निर्बाध रूप से जुड़ें।
- **मल्टी-एलएलएम समर्थन**: उन्नत एआई क्षमताओं के लिए विभिन्न बड़े भाषा मॉडल का उपयोग करें।
- **ट्रेडिंग कौशल अनुकूलन**: व्यक्तिगत प्राथमिकताओं के अनुरूप ट्रेडिंग रणनीतियों को निजीकृत और अनुकूलित करें।
- **बहुभाषी यूआई**: स्वचालित भाषा पहचान के साथ, 42 भाषाओं में उपलब्ध यूजर इंटरफेस का आनंद लें।
- **बैकअप/रिस्टोर**: अपने एप्लिकेशन डेटा और सेटिंग्स का सुरक्षित रूप से बैकअप लें और पुनर्स्थापित करें।
- **डेवलपर जानकारी और मार्गदर्शिकाएँ**: डेवलपर्स और उन्नत उपयोगकर्ताओं के लिए विस्तृत जानकारी और मार्गदर्शिकाओं तक पहुंचें।

## टेक स्टैक

- **फ्रेमवर्क**: रिएक्ट नेटिव + एक्सपो + टाइपस्क्रिप्ट
- **नेविगेशन**: एक्सपो राउटर
- **राज्य प्रबंधन**: प्रतिक्रिया संदर्भ एपीआई
- **स्थानीय भंडारण**: AsyncStorage
- **प्लेटफ़ॉर्म**: आईओएस और एंड्रॉइड
- **भाषा समर्थन**: 42 भाषाएँ (डिफ़ॉल्ट फ़ॉलबैक: थाई)

## स्थापना

कनुत्सनानएफएक्स के साथ आरंभ करने के लिए, इन चरणों का पालन करें:

1. **रिपॉजिटरी को क्लोन करें**:
```bash
गिट क्लोन https://github.com/kanutsanan1988/KanutsananFX.git
सीडी कनुत्सनानएफएक्स
```

2. **निर्भरताएं स्थापित करें**:
```bash
npm install
# या पीएनपीएम का उपयोग कर रहे हैं
pnpm install
```

## ऐप चलाना

एप्लिकेशन को विकास मोड में चलाने के लिए:

```bash
npx expo start
```

इससे आपके ब्राउज़र में एक्सपो डेव टूल्स खुल जाएगा।फिर आप ऐप को यहां चलाना चुन सकते हैं:

- **एंड्रॉइड**: टर्मिनल में `ए` दबाएं या क्लिक करें
एंड्रॉइड डिवाइस/एमुलेटर पर चलाएं।
- **आईओएस**: टर्मिनल में `i` दबाएं या `आईओएस सिम्युलेटर पर चलाएं` पर क्लिक करें।

वैकल्पिक रूप से, आप सीधे एक विशिष्ट प्लेटफ़ॉर्म पर चला सकते हैं:

```bash
npm run android
npm run ios
```

## एंड्रॉइड और आईओएस के लिए बिल्डिंग

विभिन्न प्लेटफार्मों और उद्देश्यों के लिए एप्लिकेशन बनाने के निर्देश यहां दिए गए हैं:

### एंड्रॉइड

- **पूर्वावलोकन एपीके (परीक्षण के लिए)**:
```bash
npm run build:android:apk
```

- **प्रोडक्शन एएबी (गूगल प्ले स्टोर के लिए)**:
```bash
npm run build:android:production
```

### आईओएस

- **पूर्वावलोकन बिल्ड (डिवाइस पर परीक्षण के लिए)**:
```bash
npm run build:ios
```

- **सिम्युलेटर बिल्ड (आईओएस सिम्युलेटर पर परीक्षण के लिए)**:
```bash
npm run build:ios:simulator
```

- **प्रोडक्शन बिल्ड (एप्पल ऐप स्टोर के लिए)**:
```bash
npm run build:ios:production
```

## कैसे उपयोग करें

कनुत्सनानएफएक्स आपकी विदेशी मुद्रा व्यापार गतिविधियों के प्रबंधन के लिए एक सहज ज्ञान युक्त इंटरफ़ेस प्रदान करता है।ऐप लॉन्च करने के बाद, आप यह कर सकते हैं:

- वास्तविक समय में बाजार की जानकारी प्राप्त करने और ट्रेडिंग कमांड निष्पादित करने के लिए **एआई चैट एजेंट** का उपयोग करें।
- त्वरित और कुशल व्यापार प्रबंधन के लिए **4 ट्रेडिंग कमांड** तक पहुंचें।
- अपनी रणनीतियों को अनुकूलित करने के लिए सेटिंग्स में अपने **ट्रेडिंग कौशल** को अनुकूलित करें।
- **बैकअप/रिस्टोर** कार्यक्षमता के साथ अपना खाता और डेटा प्रबंधित करें।

## बहुभाषी समर्थन

कनुत्सनानएफएक्स व्यापक बहुभाषी समर्थन प्रदान करता है, जिससे यह वैश्विक दर्शकों के लिए सुलभ हो जाता है:

- **42 भाषाओं** का समर्थन करता है।
- ऐप लॉन्च पर **स्वचालित डिवाइस भाषा का पता लगाना**।
- इन-ऐप सेटिंग से **मैन्युअल रूप से अपनी पसंदीदा भाषा चुनने** का विकल्प।
- **थाई** को डिफ़ॉल्ट फ़ॉलबैक भाषा के रूप में सेट किया गया है।
