# ਕਨੂਤਸਨਾਨਐਫਐਕਸ - ਫਾਰੇਕਸ ਟਰੇਡਿੰਗ ਏਆਈ ਏਜੰਟ ਐਪਲੀਕੇਸ਼ਨ

## ਸੰਖੇਪ ਜਾਣਕਾਰੀ

ਕਨੂਤਸਨਾਨਐਫਐਕਸ ਇੱਕ ਅਤਿ-ਆਧੁਨਿਕ ਫਾਰੇਕਸ ਟਰੇਡਿੰਗ ਏਆਈ ਏਜੰਟ ਐਪਲੀਕੇਸ਼ਨ ਹੈ ਜੋ ਵਪਾਰੀਆਂ ਨੂੰ ਬੁੱਧੀਮਾਨ ਆਟੋਮੇਸ਼ਨ ਨਾਲ ਸ਼ਕਤੀ ਪ੍ਰਦਾਨ ਕਰਨ ਲਈ ਤਿਆਰ ਕੀਤੀ ਗਈ ਹੈ। **ਐਕਸਪੋ / ਰੀਐਕਟ ਨੇਟਿਵ** ਨਾਲ ਬਣੀ, ਇਹ ਐਪਲੀਕੇਸ਼ਨ **ਆਈਓਐਸ ਅਤੇ ਐਂਡਰਾਇਡ** ਦੋਵਾਂ ਪਲੇਟਫਾਰਮਾਂ ਲਈ ਨਿਰਵਿਘਨ ਸਹਾਇਤਾ ਪ੍ਰਦਾਨ ਕਰਦੀ ਹੈ, ਜੋ ਡਿਵਾਈਸਾਂ ਵਿੱਚ ਇੱਕ ਇਕਸਾਰ ਅਤੇ ਸ਼ਕਤੀਸ਼ਾਲੀ ਵਪਾਰਕ ਅਨੁਭਵ ਨੂੰ ਯਕੀਨੀ ਬਣਾਉਂਦੀ ਹੈ। ਐਪ ਵਿੱਚ ਹੁਣ **42 ਭਾਸ਼ਾਵਾਂ ਲਈ ਵਿਆਪਕ ਬਹੁ-ਭਾਸ਼ਾਈ ਸਹਾਇਤਾ** ਹੈ, ਜਿਸ ਵਿੱਚ ਆਟੋਮੈਟਿਕ ਡਿਵਾਈਸ ਭਾਸ਼ਾ ਖੋਜ ਅਤੇ ਉਪਭੋਗਤਾਵਾਂ ਲਈ ਸੈਟਿੰਗਾਂ ਦੇ ਅੰਦਰ ਆਪਣੀ ਪਸੰਦੀਦਾ ਭਾਸ਼ਾ ਨੂੰ ਹੱਥੀਂ ਚੁਣਨ ਦੀ ਲਚਕਤਾ ਸ਼ਾਮਲ ਹੈ।


## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

*   **ਏਆਈ ਚੈਟ ਏਜੰਟ:** ਸੂਝ ਪ੍ਰਾਪਤ ਕਰਨ ਅਤੇ ਆਪਣੇ ਵਪਾਰ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰਨ ਲਈ ਇੱਕ ਬੁੱਧੀਮਾਨ ਏਆਈ ਨਾਲ ਗੱਲਬਾਤ ਕਰੋ।
*   **4 ਵਪਾਰਕ ਕਮਾਂਡਾਂ:** ਐਪ ਰਾਹੀਂ ਸਿੱਧੇ ਜ਼ਰੂਰੀ ਵਪਾਰਕ ਕਾਰਵਾਈਆਂ ਕਰੋ।
*   **ਮੈਟਾਏਪੀਆਈ ਏਕੀਕਰਣ:** ਰੀਅਲ-ਟਾਈਮ ਵਪਾਰ ਲਈ ਮੈਟਾਏਪੀਆਈ ਰਾਹੀਂ ਮੈਟਾਟ੍ਰੇਡਰ ਖਾਤਿਆਂ ਨਾਲ ਨਿਰਵਿਘਨ ਜੁੜੋ।
*   **ਮਲਟੀ-ਐਲਐਲਐਮ ਸਹਾਇਤਾ:** ਵਧੀ ਹੋਈ ਏਆਈ ਸਮਰੱਥਾਵਾਂ ਲਈ ਵੱਖ-ਵੱਖ ਵੱਡੇ ਭਾਸ਼ਾਈ ਮਾਡਲਾਂ ਦਾ ਲਾਭ ਉਠਾਓ।
*   **ਵਪਾਰਕ ਹੁਨਰ ਅਨੁਕੂਲਤਾ:** ਆਪਣੀਆਂ ਤਰਜੀਹਾਂ ਨਾਲ ਮੇਲ ਕਰਨ ਲਈ ਵਪਾਰਕ ਰਣਨੀਤੀਆਂ ਨੂੰ ਨਿੱਜੀ ਬਣਾਓ ਅਤੇ ਵਧੀਆ ਬਣਾਓ।
*   **ਬਹੁ-ਭਾਸ਼ਾਈ UI:** ਵਿਸ਼ਵਵਿਆਪੀ ਪਹੁੰਚਯੋਗਤਾ ਲਈ 42 ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਉਪਲਬਧ ਇੱਕ ਉਪਭੋਗਤਾ ਇੰਟਰਫੇਸ।
*   **ਬੈਕਅੱਪ/ਰੀਸਟੋਰ:** ਆਪਣੀ ਐਪਲੀਕੇਸ਼ਨ ਡੇਟਾ ਅਤੇ ਸੈਟਿੰਗਾਂ ਦਾ ਸੁਰੱਖਿਅਤ ਢੰਗ ਨਾਲ ਬੈਕਅੱਪ ਲਓ ਅਤੇ ਰੀਸਟੋਰ ਕਰੋ।
*   **ਡਿਵੈਲਪਰ ਜਾਣਕਾਰੀ ਅਤੇ ਗਾਈਡਾਂ:** ਡਿਵੈਲਪਰਾਂ ਅਤੇ ਉਪਭੋਗਤਾਵਾਂ ਲਈ ਵਿਆਪਕ ਸਰੋਤ।


## ਤਕਨੀਕੀ ਸਟੈਕ

*   **ਪ੍ਰੋਜੈਕਟ ਦਾ ਨਾਮ:** ਕਨੂਤਸਨਾਨਐਫਐਕਸ
*   **ਫਰੇਮਵਰਕ:** ਰੀਐਕਟ ਨੇਟਿਵ + ਐਕਸਪੋ + ਟਾਈਪਸਕ੍ਰਿਪਟ
*   **ਨੈਵੀਗੇਸ਼ਨ:** ਐਕਸਪੋ ਰਾਊਟਰ
*   **ਸਟੇਟ ਪ੍ਰਬੰਧਨ:** ਰੀਐਕਟ ਕੰਟੈਕਸਟ ਏਪੀਆਈ
*   **ਸਥਾਨਕ ਸਟੋਰੇਜ:** ਅਸਿੰਕਸਟੋਰੇਜ
*   **ਪਲੇਟਫਾਰਮ:** ਆਈਓਐਸ ਅਤੇ ਐਂਡਰਾਇਡ
*   **ਭਾਸ਼ਾ ਸਹਾਇਤਾ:** 42 ਭਾਸ਼ਾਵਾਂ
*   **ਮੂਲ ਫਾਲਬੈਕ ਭਾਸ਼ਾ:** ਥਾਈ


## ਇੰਸਟਾਲੇਸ਼ਨ

ਕਨੂਤਸਨਾਨਐਫਐਕਸ ਨਾਲ ਸ਼ੁਰੂਆਤ ਕਰਨ ਲਈ, ਇਹਨਾਂ ਕਦਮਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ:

1.  ਰਿਪੋਜ਼ਟਰੀ ਕਲੋਨ ਕਰੋ (ਜੇ ਲਾਗੂ ਹੋਵੇ)।
2.  npm ਜਾਂ pnpm ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਨਿਰਭਰਤਾਵਾਂ ਸਥਾਪਿਤ ਕਰੋ:

    ```bash
    npm install
    # ਜਾਂ
    pnpm install
    ```


## ਐਪ ਚਲਾਉਣਾ

ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਡਿਵੈਲਪਮੈਂਟ ਮੋਡ ਵਿੱਚ ਚਲਾਉਣ ਲਈ:

```bash
npx expo start
```

ਕਿਸੇ ਖਾਸ ਪਲੇਟਫਾਰਮ 'ਤੇ ਚਲਾਉਣ ਲਈ:

```bash
npm run android
# ਜਾਂ
npm run ios
```


## ਐਂਡਰਾਇਡ ਅਤੇ ਆਈਓਐਸ ਲਈ ਬਿਲਡ ਕਰਨਾ

ਐਪਲੀਕੇਸ਼ਨ ਨੂੰ ਵੱਖ-ਵੱਖ ਐਂਡਰਾਇਡ ਅਤੇ ਆਈਓਐਸ ਟੀਚਿਆਂ ਲਈ ਬਿਲਡ ਕਰਨ ਲਈ ਕਮਾਂਡਾਂ ਹੇਠਾਂ ਦਿੱਤੀਆਂ ਗਈਆਂ ਹਨ:

### ਐਂਡਰਾਇਡ

*   **ਪੂਰਵਦਰਸ਼ਨ ਏਪੀਕੇ:**
    ```bash
    npm run build:android:apk
    ```
*   **ਉਤਪਾਦਨ ਏਏਬੀ:**
    ```bash
    npm run build:android:production
    ```

### ਆਈਓਐਸ

*   **ਪੂਰਵਦਰਸ਼ਨ ਬਿਲਡ:**
    ```bash
    npm run build:ios
    ```
*   **ਸਿਮੂਲੇਟਰ ਬਿਲਡ:**
    ```bash
    npm run build:ios:simulator
    ```
*   **ਉਤਪਾਦਨ ਬਿਲਡ:**
    ```bash
    npm run build:ios:production
    ```


## ਕਿਵੇਂ ਵਰਤਣਾ ਹੈ

ਵਿਸਤ੍ਰਿਤ ਵਰਤੋਂ ਨਿਰਦੇਸ਼ ਐਪਲੀਕੇਸ਼ਨ ਦੀਆਂ ਗਾਈਡਾਂ ਦੇ ਅੰਦਰ ਪ੍ਰਦਾਨ ਕੀਤੇ ਜਾਣਗੇ। ਆਮ ਤੌਰ 'ਤੇ, ਐਪ ਨੂੰ ਸਥਾਪਿਤ ਕਰਨ ਅਤੇ ਚਲਾਉਣ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਆਪਣਾ ਮੈਟਾਏਪੀਆਈ ਕਨੈਕਸ਼ਨ ਕੌਂਫਿਗਰ ਕਰ ਸਕਦੇ ਹੋ, ਵਪਾਰਕ ਹੁਨਰਾਂ ਨੂੰ ਅਨੁਕੂਲਿਤ ਕਰ ਸਕਦੇ ਹੋ, ਅਤੇ ਆਪਣੀਆਂ ਫਾਰੇਕਸ ਵਪਾਰਕ ਗਤੀਵਿਧੀਆਂ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰਨ ਲਈ ਏਆਈ ਚੈਟ ਏਜੰਟ ਨਾਲ ਗੱਲਬਾਤ ਕਰ ਸਕਦੇ ਹੋ।


## ਬਹੁ-ਭਾਸ਼ਾਈ ਸਹਾਇਤਾ

ਕਨੂਤਸਨਾਨਐਫਐਕਸ ਮਜ਼ਬੂਤ ਬਹੁ-ਭਾਸ਼ਾਈ ਸਮਰੱਥਾਵਾਂ ਦੀ ਪੇਸ਼ਕਸ਼ ਕਰਦਾ ਹੈ, ਜੋ 42 ਵੱਖ-ਵੱਖ ਭਾਸ਼ਾਵਾਂ ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ। ਐਪਲੀਕੇਸ਼ਨ ਤੁਹਾਡੀ ਡਿਵਾਈਸ ਦੀ ਭਾਸ਼ਾ ਨੂੰ ਸਮਝਦਾਰੀ ਨਾਲ ਖੋਜਦੀ ਹੈ ਅਤੇ UI ਨੂੰ ਆਪਣੇ ਆਪ ਵਿਵਸਥਿਤ ਕਰਦੀ ਹੈ। ਤੁਸੀਂ ਇਨ-ਐਪ ਸੈਟਿੰਗਜ਼ ਮੀਨੂ ਰਾਹੀਂ ਕਿਸੇ ਵੀ ਪਸੰਦੀਦਾ ਭਾਸ਼ਾ ਵਿੱਚ ਆਸਾਨੀ ਨਾਲ ਬਦਲ ਸਕਦੇ ਹੋ।
